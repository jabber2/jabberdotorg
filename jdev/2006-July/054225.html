> Do i need jabber account in order to interact with AOL/Yahoo/MSN?
yes. 
you need:
1) Setup jabber server with gateways. Almost any server will go. My personal 
preference is ejabberd. Gateways are additional components and installed and 
confugured separately. If someone will allow you to use his server that 
matches required config - you can skip this step.
2) Create jabber account on this server, aol account, yahoo account, msn 
account.
3) Register aol/yim/msn accounts for this jabber account
4) go online and send messages

I think you can find all this info in documentation publisher on jabber.org 
(or try google).
