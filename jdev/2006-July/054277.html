You can put that kind of information, disclaimer, TOS, etc., in the
registration form which your server returns to the client which is going to
create a account. If that isn't supported the server can just provide Off-Band
Registration where the user can read all that info before applying.
http://www.jabber.org/jeps/jep-0077.html

For users who already have an account most server admins just send a message
to all users on changes on that topic.
