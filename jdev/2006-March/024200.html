I wrote a Perl-based Script about a year ago which is capable of all
features you asked for:
http://jabberstudio.org/projects/webstatus/project/view.php

The Script is only available in the CVS, because that is the old
version, connecting as a client and really a pain to set it up -
however, it is working fine and serving the users at jabber.ccc.de
since some time now. It provides about 10 different themes, and small
dynamic website showing the status in verbose.

Since I am now employed with mabber (http://www.mabber.com), and
being therefore paid for doing nice stuff with Jabber, I started
to rewrite the whole script as a Component which should be available
as a first preview release within the next three weeks.
