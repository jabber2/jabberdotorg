I'm proud to announce the first public release of JSJaC. JSJaC is a
jabber client library written in JavaScript to ease implementation of
web-based jabber clients. For communication with a jabber server it
needs to support either HTTP Polling or HTTP Binding. JSJaC has an
object oriented design which should be quite easy to use. Communication
is done by using the XML HTTP Request object. Your browser must support
this. You can read more about JSJaC at its homepage located at 

http://jsjac.jabberstudio.org/

-Steve
