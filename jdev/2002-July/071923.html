Patrick Fitzgerald wrote:
> Is it is possible for different processes on the same machine to log

> as the same user, but different resources, for example,
>    patrickf at jabber.com/work

> or even
>    patrickf at jabber.com/work/phone

This is very possible (I do it on a regular basis). I've also done it with
Winjab and JabberIM (both of which use JabberCOM). So I know the library can
do this properly. It may have something to do w/ how you are implementing
it. I've never seen/had any problems if the 2 resources are in different
processes.
