> So which is the preferred method?

Component accept. The library load method isn't recommended anymore,
mostly for performance reasons, and only method supported by the next
version of the server (1.5) will be component accept, so library load
components won't work unless they are run via a jabberd 1.4 using the
<uplink/> technique.
