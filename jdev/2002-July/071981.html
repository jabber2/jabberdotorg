JabberCOM does not handle file transfers. Most jabber clients accomplish
this thru the use of HTTP components/libraries, and just use a
"simple-temporary" jabber server on the sender, and a simple HTTP client for
the receiver. These transfers are done out of band. Checkout the winjab
source for an example of how this works.
