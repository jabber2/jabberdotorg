Our company is interested in sponsoring someone (or a group of people) to 
build in Macromedia Flash 5/6 a library for building Jabber clients.

It would need to allow people to login/out, create accounts, send/receive 
messages, add contacts, and show/change status.  It should also include a 
demo front end.  However, it needs to be as modular as possible so that it 
can be hooked into virtually any flash movie.  JabberFlash 
(http://sourceforge.net/projects/jabberflash) might make a good starting point.

If the Jabber Foundation is already working on something like this, let me 
know, maybe we can work together.
