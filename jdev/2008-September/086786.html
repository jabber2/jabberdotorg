Whilst doing the server-side recently, we tested a lot of clients  
too. Not all clients handle invitations, and some handle them badly.


> Any thoughts or explanations? Is there an "old way" and the "XEP

There is an old direct method, which we've recently revamped for  
various reasons, but clients tend not to send those.

One of the reasons for bringing these back is because privacy lists  
and similar client-side functionality can have the effect of blocking  
MUC invites, so it's entirely possible this is (one of) the problems  
you're running into.
