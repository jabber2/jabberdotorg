Hey all,
Just wanted to let you know that I released a jabber based helpdesk  
for plone. There exists a client for plone (although it is trivial to  
write clients for anything, only xmlrpc calls are required) and a  
generic python server based on twisted.
The interesting thing about it is that you connect "agents" to your  
helpdesk which chat real time with clients that visit the helpdesk  
using their jabber account. So now, you can provide support using your  
phone;)

You can try it out on http://chatblox.com and even link your account  
there to your own plone site.

The relevant pypi packages are:
qi.jabberHelpdesk  (plone client)

If you manage a jabber server and want to run your own helpdesk server  
have a look at these packages:

qi.xmpp.client (server)
qi.xmpp.admin (server)
qi.xmpp.botfarm (server)
