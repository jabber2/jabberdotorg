ALL info/query (iq) requests have to go to a users' resource. (unless the iq is directed at a server of course, since server's don't have resources)  this is done becuase the iq is designed to be targeted at a single client, or user, if you sent it to just user at host, there is no telling where it may come up.  the Jabber server will reject any iq request that is not sent with a resource on it.
