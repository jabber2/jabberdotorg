In case you haven't been to the project's website, we now have an 
alpha release you can test.

Features:
1) Plaintext authentication with LDAP
2) Digest authentication with LDAP (if LDAP password stored in 
cleartext)
3) basic vCARD (name, telephonenumber, address)

we include build instructions and a script to help you get existing 
LDAP users registered for Jabber (we don't have registration 
working yet).
