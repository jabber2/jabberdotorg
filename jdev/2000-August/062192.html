    How are you addressing the server within WinJab?  The DNS name, or the
IP?  In either case, the name you are refering to the server as *MUST* be
either the name, configured in jserver.xml, or in the list of aliases in
jserver.xml.  It uses it's name and aliases to say 'This is who I'll accept
connections as"..
