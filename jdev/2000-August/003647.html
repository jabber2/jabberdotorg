The PowWow protocol is a client to client protocol, The server just acts as
a phone book, handing out ip addresses of the buddies to the clients.
Presence detection is done by the client "pinging" all the members of the
buddy list. A PowWow transport would need to do a lot of the presence
pinging if it were to act as a PowWow proxy client.
