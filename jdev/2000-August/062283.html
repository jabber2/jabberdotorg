Jabbernaut 0.5b3 has just been released. Go to 
http://jabbernaut.sourceforge.net/download.html to download it.

This new release features many improvments, including:

  * agent support! Now it's easy to register with ICQ-t, AIM-t etc.
  * customizable roster colors
  * improved roster - resources are visible for each roster item
  * auto away works again, and so does auto-end auto-away
  * Jabbernaut can check for new version of itself upon login
  * many other new features
  * dozens of bug fixes


If you use an older version of Jabbernaut, make sure you get this 
latest release!



Have fun,
