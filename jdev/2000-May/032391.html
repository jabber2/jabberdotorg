INSTALLATION
- ------------
1) Unpack the system
	tar -zxvf groupchat-transport-x.x.x.tar.gz

	./configure && make && make install

3) Edit the groupchat.xml and set the server name to what you would
   like it to be.

4) Decide to either use the dso or socket version to connect to
etherx
	dso:
	-  Edit your etherxd.xml and add:
	<dso init='gc_init'>/path/to/your/gc_trans.so</dso>

	socket:
	-  Run the groupchat program to get the command line
