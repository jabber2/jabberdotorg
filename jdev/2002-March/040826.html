I think you should use "generic" librairies, with no Jabber stuff. Jabber
transports must manage many opened sessions , do a lot of rewriting stuff...
Things that you probably don't need.

You should look at these URLs, they should be a good starting point :

gAIM protocols implementation :
http://cvs.sourceforge.net/cgi-bin/viewcvs.cgi/gaim/gaim/src/protocols/


For yahoo : http://sourceforge.net/projects/libyahoo/

For ICQ : http://kicq.sourceforge.net/icqlib.shtml
