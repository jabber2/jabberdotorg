First of all, init.c is needed for Windows version of transports only, so I
supose you intend to compile for windows.

Next. The init.o file is the same for all transports so if you have compiled
another transport is enough to copy that file into MSN transport directory.
Third if you don't have init.c file you should find it in other transports
(for Windows). Of not, you can mail me and i'll mail it to you.

Hope it helps.
