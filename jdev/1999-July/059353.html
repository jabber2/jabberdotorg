Here is what I've been thinking:

Based on the "team" model I've been working on for the tools and such, I
want to create a special "marketing" or "showme" team.  This team would be
responsible for coming up with a Logo(likely via a contest they can hold),
the structure/layout of the web site, press releases, introductory
material, info at jabber.org, etc...  There are quite a few people that have
already volunteered for this type of work, getting all of them together
and moving forward would be a good thing.

As far as the icons go, the same team could come up with a reccomended set
of icons for clients to use.  I've always also wanted to be able to have
status icons change dynamically, so based on the upcoming file transfer
proposal I'm writing, status packets could contain a reference to a new
status icon to download in the <icon></icon> field.

This is definately something that needs to get moving quickly here... I'll
include this team in my list of teams to form next week.
