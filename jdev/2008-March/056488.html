agsXMPP 1.0 released

What is agsXMPP?
agsXMPP is a SDK / library for the eXtensible Messaging and Presence 
Protocol (XMPP) protocol written in managed C# dedicated to .NET and 
Mono technologies.

Its designed for:
* Microsoft .NET Framework (1.1 - 3.5)
* Microsoft .NET Compact Framework (Pocket PC, Smartphone, Windows CE)
* Mono

The SDK is released as open source under a dual license.

You can download agsXMPP 1.0 at:
http://www.ag-software.de/index.php?page=download

You can find more info about agsXMPP here:
http://www.ag-software.de/index.php?page=agsxmpp-sdk
