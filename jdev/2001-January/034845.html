Ah yes..

WinJab/JabberIM will both send update requests to j.org in an effort
to check for later versions. This is an intentional thing that you can
always have the latest & greatest. :) 

That said, there should definately be a way to turn off update requests
in the clients -- you should talk with Peter Millard (WinJab author) about
adding a configurable switch for that.
