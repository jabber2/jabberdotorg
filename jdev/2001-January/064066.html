Hello all

well, I've put together a rag bag of stuff that contains a 
tentative alpha version of the groupchat(-only, for now) client
I'm developing.

It's called 'sjabber' and is modelled after sirc, the great 
text-based IRC client.

This started out as a scratch to an itch - I use dumb terminals
a lot at home (yes, I'm a collector ;-) and wanted to lurk/take
part in the jdev conference, so needed a console-based way of
doing that. 

I'm releasing it in case anyone's interested; it's very rough
round the edges but the basic conference functionality is there.

Please read the README fully first. 

It's at 

http://www.pipetree.com/~dj/sjabber-0.2.tar.gz

All comments and questions welcome, to me.
