If you are planning to use only a thousand users you should be fine, as far 
as I know there is no real loadbalancing, although the commercial jabber.com 
side supposedly will offer that.

For a thousand or so users, no worries :-)

Dennis

PS Real limits are messages/second. An idle user only takes up a few kb of 
memory, but 2 users sending 100 messages/second will hurt the server :-)
