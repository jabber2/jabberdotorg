Hello,

The icq-transport module in CVS is working fairly well.  I've
received some reports of Bus Errors on Solaris and BSD, and
I'm working on fixing this bug.  BTW, if anyone out there
happens to get a Bus Error it would help immensely if you could
e-mail the back trace of the core dump to sheath at jabber.org.

This version of icq-transport is new and has been in CVS
less then two weeks.  So if you've had issues with previous
versions you might want to give this one a shot.  Also, I'm
planning on releasing a beta version once 1.4 is out.

Schuyler
