The short answer is no. Jabber.com was working on a win32 port but that
project has been deprecated in favor of scalability work and the like.
I've talked with a few folks in the open-source community who are
interested in doing this, but there's no code to show for it yet as far
as I know.

See also http://docs.jabber.org/no-sgml/faq.trp#platforms
