Hello Everyone,

I don’t think there is a voice support in jabber at this moment. The voice
support need to come from jabber clients. The server can act as location
server and/or proxy. I am volunteering to add voice support to jabber
clients.  If someone is already working on this, please let me know and we
can do it together. I would also appreciate group’s input which client is of
most interest and can be extended. For portability issues I recommend, using
Java based client. I think adding voice to jabber is a cool idea. There are
a number of proposed standards for signaling and negotiations of voice
codecs. SIP(Session Initiation Protocol) IETF RFC2543 is the proposed
standard. I know there has been some work done on using SIP for Instant
Messaging. If someone has different opinion or options, please let me know.
