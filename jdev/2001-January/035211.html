Output from this series of commands would be an excellent starting point (Note,
I don't really need to see if cvs failed because it's installed just the build
;-] ):

cvs -z3 -d:pserver:anoncvs at jabber.org:/home/cvs login
password:  anoncvs
cvs -z3 -d:pserver:anoncvs at jabber.org:/home/cvs co jabber2
cd jabber2
./configure
make

It is highly suggested that you use a GNU compiler and GNU make system, but it
should work otherwise.  If you want to try it with and without GNU systems, that
would be greatly appreciated as well.
