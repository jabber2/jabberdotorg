1.4 Release Inching Closer

I apologize that 1.4 hasn't been released yet, and we're all working
very hard to make it happen as soon as possible.  The good news is
that there haven't been any major bugs found in pre2 and the current
version running @jabber.org has been quite stable.

The reason it's not released yet is that due to popular interest, the
conferencing service is being cleaned up and prepared for shipment as
a default component with 1.4.  I'm close to finishing it and then
tracking down a few minor issues in other peices prior to release.

So watch for a Release Candidate in the next 1-2 days, and after a few
days of thurough testing and cross-platform validation, we will have
the best server release ever! :)
