Lars,

I had good success on Sol 5.8 .... The biggest hurdle I found was that you
will need to get gnu-make -- the latest and greatest gcc and lots of
patience. Also make sure that you have the "right" versions of code to work
with -- as for the Macros -- yes you will need to pull a copy from CVS.
Also make sure you use gnu tar to unpack these -- Solaris' tar on 5.8 is
hopelessly brain dead.

Enjoy

BN
