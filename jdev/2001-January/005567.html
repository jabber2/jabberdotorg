Refer to these listing in the archives.

http://mailman.jabber.org/pipermail/jdev/2001-January/004523.html
http://mailman.jabber.org/pipermail/jdev/2000-November/003670.html

Also, make the following changes to jabber.xml
  <service id='icqtrans.mycompany.com'>
    <accept>
      <ip>xxx.xxx.xxx.xxx</ip>
      <port>9000</port>
      <secret>test</secret>
    </accept>
  </service>


        <agent jid="icqtrans.mycompany.com">
          <name>ICQ Transport</name>
          <description>This is the ICQ Transport</description>
          <transport>ICQ#</transport>
          <service>icq</service>
          <register/>
        </agent>

To start icq, type

icqtransport

The port #9000 is HARDCODED.
