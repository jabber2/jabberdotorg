Re-posting here, new features:

* Implements adhoc based Account associated Client Certificates' 
Management (mod_adhoc_cm)
* Add cleanli.st API usage to do verification on supplied E-Mail 
addresses to the OOB Register API (mod_register_json)
* Majorily simplify the MUC Custon Configuration API, and make it event 
based
JSON Internal Storage (experimental) - Allow to use JSON serialization 
optionally for flat files
* Allow to export the PEP & PubSub service objects via events

The full changelog can be found here: 
http://lightwitch.org/blogger/maranda/2014/04/metronome-im-v35-released.
