BTW, all the hosting code (building packages, docker images, deploying) is
at https://github.com/buddycloud/hosting. Pull requests welcomed.

The hosted environment is designed for developers that want to add a social
layer to their XMPP stack - Buddycloud channels, media sharing, push
notifications to mobile apps, importing the social graph from Facebook or
Twitter and search and recommendations.

The whole stack is open source, the hosted environment simply enables
running Buddycloud without maintaining a full stack of components.

We'd love your feedback and issues and feature requests (
https://github.com/buddycloud/hosting/issues)
