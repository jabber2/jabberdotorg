| is there any Odigo transport or anybody can give me any description of

Odigo uses a proprietary protocol that is not documented. As far as I
know there are no sources of how the protocol works.

If you want to write a transport, you might want to hook into the Odigo
network by their IMIP server. I made some tests with this Odigo access.
My results can found on http://matthias.wimmer.name/bunt/?seite=imu-en.xml
