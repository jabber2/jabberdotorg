I don't know of anyone that has implemented server-to-server connectivity
between Jabber and Wireless Village.....both have their own s2s protocols.
This would be an interesting exercise, although there are very few if any
Wireless Village servers deployed in a production mode today.   I know of at
least one implementation of a gateway that allows Wireless Village clients
to connect to a Jabber server (from France Telecom R&D).

There are a number of wireless Jabber clients available from companyies like
mov software (http://www.movsoftware.com/), tipic (http://www.tipic.com/)
and antepo (www.antepo.com/).
