Subject: [Registrar-announce] updated disco categories

I have added two new categories to the official registry for Service
Discovery: "client" and "server". The only defined type for the "server"
category is "im" (so far). There are 6 defined types for clients:

- bot
- console
- handheld
- pc
- phone
- web

I have also removed the "pim" category and its single type ("vcard"),
since as far as I know this category does not map to any existing
or planned entities on the Jabber network.

The updated information is here:

http://www.jabber.org/registrar/disco-categories.html
