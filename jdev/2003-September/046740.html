I've talked on and off over the last year with folks from MIT's DotLRN
project about possible Jabber/XMPP integration into their effort. Given
the recent interest in Jabber/XMPP within the Internet2 community, it
seems that such integration might be even more attractive now. DotLRN is
built on OpenACS and the AOLserver webserver, which despite its name is 
open source (both of these are built in Tcl). One of the DotLRN folks 
just poked me via Jabber about the possibility of building a Jabber 
server in Tcl using tDOM as a way to pursue integration of Jabber support 
into DotLRN. If you are interested (it's *possible* there might be 
funding involved), please ping me via Jabber.

Here are some relevant URLs:

http://dotlrn.mit.edu/
http://openacs.org/
http://www.aolserver.com/
http://www.tdom.org/

Thanks!
