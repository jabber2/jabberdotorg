My understanding is that Reuters does not use a modified MSN Messenger
system, but is the beta customer for MS's new "SIP-based" server, which
used to be called RTC but has been renamed. So anything related to the
Reuters system is totally unrelated to the MSN Messenger service. As to
the AIM connectivity, I would think this is some sort of gateway, since
MS's SIP-based system doesn't even support server-to-server.
