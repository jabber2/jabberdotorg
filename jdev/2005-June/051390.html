Yes, Jabber support rooms through use of the Multi User Chat Extension
(see JEP 45: http://www.jabber.org/jeps/jep-0045.html). Permanent and
temporary rooms are accepted.

For the rating, I think some module could be created using pubsub
support (JEP 60: http://www.jabber.org/jeps/jep-0060.html).
