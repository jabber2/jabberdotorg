Hi!


Is it intended, that jabberd 1.4.3 will become the first jabberd that
crashes with the default configuration file? Or maybe we want users
invite to upgrade to jabberd version 2?

Else I don't understand why new features are added to this version just
before it is released that seem to be untested.

mod_auth_crypt crashes the server for each new user that registers with
it because it contains an undefined symbol: crypt. As the manual of the
crypt() function states, programs using this function have to be linked
with libcrypt ... but jabberd isn't.
