The jabberd project team is pleased to announce the release of jabberd
2.0 beta 2. This is the first in a series of beta releases of the
upcoming jabberd 2 server.

The server is a complete rewrite, with none of the old 1.4 code
included. This makes the transition non-trivial. A tool is included with
this release to assist with migrating users from 1.4 servers.

The codebase is now in feature freeze, and so is suitable for testing
and planning. It shouldn't be used in a production environment yet.

Changes since b1:

    * 1.4 migration tool [rob]
    * Configurable router reconnect [rob]
    * Lots of memory corruption and other bugs fixed [rob]

Downloads are available here:

  http://www.jabberstudio.org/projects/jabberd2/releases/view.php?id=335

Special thanks to Jacek Konieczny and Paul Curtis for their help
tracking down and fixing bugs in this release - some serious stuff has
been squished this time around. Thanks guys :)

Bug reports and feature requests should be submitted using the tools on
jabberstudio.org. Anything else should be sent to jabberd at jabberstudio.org.

Have fun!

Rob.
