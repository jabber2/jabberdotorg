Again, I see this as an implementation issue. Logs are long-lasting
archives of activity within the room (we have logs for the chat service
on jabber.org going back to 2000 or 2001). Which log does the user want?
Today's? Yesterday's? The one from 2001-01-16?

I don't think this should be in the MUC spec. That spec is big enough as
it is right now. But another spec for interacting with room logs might
be good (should it be rolled into the chat history discussion?).

Peter
