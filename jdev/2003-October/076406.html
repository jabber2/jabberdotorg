Hi Peter,

  I've been thinking about this, and whilst initially it seemed more of
a protocol issue (i.e. do we want a standardized way of handling the
request), i've changed my opinion whilst writing this email. 

  My change was when I tried to work out how such an option would best
fit into the existing system. My feeling is that it would be best to
simply add an option to the room configuration form, which would allow
an owner to clear the logs. Since this is a change to the room, it makes
sense to have it here, as the users can then be informed that the
configuration has changed.

  So i've added it to my MU-Conference todo list, and hopefully there
will be a new release soon.

Regards,

  David
