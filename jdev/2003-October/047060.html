Hi!

Just released a new version of Loudmouth, this time it was versioned
0.14.

What is Loudmouth:
------------------
Loudmouth is a small C library for writing Jabber clients written using
GLib. It's designed to be extensible while maintaining ease of use. 

News in 0.14:
-------------
Loudmouth 0.14 contains a couple of fixes, such as a crash fix when
using GnuTLS, memleak fix when destroying a connection and a race
condition when using blocking methods. It also contains support for IPv6
provided by Sjoerd Simons. 

Thanks to:
----------
* Sjoerd Simons for his work on IPv6 support.
* Martyn Russell for patch to fix a problem when using GnuTLS.
* Vince Negri for build instructions when building on Windows.

Enjoy,
