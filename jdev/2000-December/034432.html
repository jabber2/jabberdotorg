Jabber.Org is pleased to invite the Jabber development commmunity
to our first public gathering.  The topics for discussion at this gathering
include:

- The Website - A new Jabber.Org is soon to be unveiled, and discussion will
                surround the new site and last minute changes and decisions.
- Protocol Extensions - A document describing new protocol extensions, and
                       refinement of some older ones has been completed, and
                       will be discussed.
- Transports - Where we stand, where they need to go, and development for
the
               future.
- Jabber Server 1.4 - The new server is almost upon us, and now is a good
time
                      to learn what's changing.

We hope to see you in attendance 10pm Central in the jdev
(jid: jdev at conference.jabber.org) conference channel.
