As many JDEVers have no doubt noticed, we have performed some major
surgery on the jabber.org website. Actually, we now have three main
sites:

1. www.jabber.org - high-level info about the project, mainly intended
for people who want a quick overview

2. dev.jabber.org - the "dev zone", containing project-related
information and resources for Jabber developers

2. docs.jabber.org - documentation

We've really just begun working on dev and docs, and will be adding
additional features to these areas soon - examples include tools for
managing Jabber projects and the ability to comment on documentation. If
you have features you'd like to see, let us know! And if you run a
project but don't see information about it on the new site, contact me
so I can add it back in.

BTW, this work was performed mainly by Justin Mecham, Peter Millard,
Jeremie Miller, and me ... just so you know where to send the praise or
blame. :)
