From: <David_Antell at infoimage.com>
Subject: [JDEV] Jabberd 1.2 and HTTP:
> I have jabber 1.2 running and am looking into the HTTP portion to try and
> write an .asp based client. Has support been removed in 1.2 for HTTP
> connects? If so is it scheduled for a comeback?

    Support hasn't been 'removed' really, but the underlying infrastructure
has changed dramatically, thus the mod_http module suffers from code rot to
the extent where it has to pretty much be redone to support 1.2.  Alot of
the underlying 'code' will be able to be reused, but the actual server
interaction has to be redone.

  The good news is, the underlying architecure has been rewritten so that
support of an alternate server system can be integrated much easier.
Basically, it's just another session manager, really..
