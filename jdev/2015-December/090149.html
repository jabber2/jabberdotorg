Hi XMPP folks!

We are pleased to announce the release of Libervia (Salut à Toi) 0.6.0.

Salut à Toi is a Multipurpose, multi frontend, free (as in freedom) 
decentralized communication tool, and Libervia is its web interface.

This release is a result of a long work on the blogging/microblogging (based 
on PubSub/XEP-0277): Libervia feature a blogging engine with public or 
restricted access, fully based on XMPP. This allows us to communicate with 
other projects such as Movim or Jappix.

We have also implemented Jingle and use it for file transfer.

Beside these two majors features, there have been a lot of various 
improvements (more than 15 new plugins!). The project should still considered 
as "alpha" quality, but this was the last release before our "general 
audience" release, which will focus on stability.

We have also a running (close to be finished) crowd-funding campaign, to 
support the development of a new desktop interface, and an Android port.

Any feedback welcome!

official website: http://salut-a-toi.org
web frontend demo: https://libervia.org
crowd-funding campaign: http://www.arizuka.com/en/projects/libervia


Thanks
