On Tue, 5 Jan 1999, Philip Bowens wrote:

> Yes, I think that is a good idea. Although some mail programs (pine) have
> an option to Reply-to-All, It would just be easier that way.
> 
> On Tue, 5 Jan 1999, Jeremie Miller wrote:
> 
> > Does anyone care if I set the Reply-To: header to point to
> > jdev at jabber.org?  It might make it easier and is fairly common on other
> > lists... 

Common, but wrong for at least one (very good) reason... namely, the
default failure modes are no where near compatable.  If the default reply
to is a person, and you fail to change it for mail to the list, the worst
that happens is you re-send.  If the default reply is to the list, and you
fail to change it to a person, you can send some very inappropriate things
to the list as a whole.  It's happened, and it's caused some very strained
situations.

Anyway, just MHO, and some advice.  FWIW.
