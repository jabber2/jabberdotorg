Hello jabbers!

I would like to know your opinion about the best way to implement an
anonymous recieve-only account.

I have the following requirements:

	Any number of users (guests) should be able to use
	the account simultaneous.

	The guest account should not be able to send
	anything except prescense information to 
	any jabber entity.

	Addressing individual users of the guest
	account by using the /resource.

	Send to all users of the guest account by
	by not supplying a /resource.


What do you think? Will this be possible to implement using the mapi? Or
will it require changes to the server? Will it break something?
