Hello,

A GUI Linux installer for ejabberd has been available for quite some 
time now on Process-one website. You can download it from:

http://www.process-one.net/en/projects/download.html

It is bundled with a stripped down version of Erlang and you do not need 
anything else on your computer.

If you have any suggestion to improve the install process and the 
configuration steps involved, you are welcome. Planned new features are 
for example SSL configuration from the install GUI. Note that the 
installer can also work directly from the console without X server.

It is only for Linux-x86 now, but other operating systems and 
architectures are planned, and will probably be developed if no problem 
is found on this version of the installer.

Cheers,
