Hello,

J-EAI 1.0 beta 5 has been released. J-EAI is an Enterprise Applications 
Integration platform based on the XMPP protocol.

This release fixes some minors bugs on pubsub and is now compliant with 
ejabberd 0.9.1. This means that J-EAI supports virtual hosting, and 
shared rosters. Documentation has been updated.

The release is now distributed with an installer for Linux. The 
installer takes care of installing all the necessary components (except 
Java, if you want to use the Java based features of J-EAI).

Everything can be downloaded from:
http://www.process-one.net/en/projects/j-eai/index.html

Hopefully, J-EAI 1.0 stable version will be published before the end of 
the month, depending on the problem

Next steps will be mainly related to the future version 1.1. Several new 
contributors have joined in, proposing patches (that will be integrated 
in version 1.1) and very interesting new features idea.
Do not hesitate to participate !
