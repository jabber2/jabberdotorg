Well, it might not (at least yet) handle *everything* for you. Normally 
what the installer does for now:
- It install the software and configure it to run from the install 
location. No additional software is needed. You just need to install the 
binary version.
- You can configure:
   . hostname,
   . admin username
   . admin password

We are very interested in knowing what you would need the installer to 
make for you. I think being able to select if you want to enable SSL/TLS 
or not is an important feature that is very likely to be added in the 
next version of the installer.
If you have any other ideas, you are welcome !
