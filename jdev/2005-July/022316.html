That depends on the server you choose. The XMPP server Jabberd2, for example, 
needs to have mu-conference embedded in JCR, to enable MUC 
(conference/rooms). On the other hand you have servers such as ejabberd 
( http://ejabberd/jabber.ru/ ) that have built-in support for rooms (a 
default enabled module).
