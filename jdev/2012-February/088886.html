On 02/23/2012 04:31 PM, jun fang wrote:
> Hi all,
> Abou Jabber-RPC extension(XEP-0009), I know how to call remote method
> but how could I get known these methods?
> Is there any mechanism like Ad-Hoc Commands extension(XEP-0050) sending
> discovering support messages?
> 
> I am trying remote control using xmpp. But I am confused by these two
> extension (RPC and Ad-Hoc Commands).
> It seems RPC do not support discover messages and Ad-Hoc Commands do not
> support parameters(instead "stages").
> Any suggestion?

You should definetely use Ad-Hoc for remote control. Ad-Hoc support
parameters, you can pass it with data forms.

Jabber-RPC should be used when porting existent XML-RPC APIs to the XMPP
transport. They have specification so there's no need to discover which
API methods exist.
