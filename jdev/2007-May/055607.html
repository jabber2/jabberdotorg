> 2. If I do use the library is it possible to package it up with a new
> Firefox extension, rather than insisting a prerequisite of having
> xmpp4moz installed?

You probably want "Multiple item packaging"
(http://developer.mozilla.org/en/docs/Multiple_Item_Packaging) which
is what I'm doing with SamePlace suite (to package xmpp4moz,
SamePlace, JabBar and JabBiff together).  It's good because you only
give away one package but it gets installed as separate packages with
proper version checking.
