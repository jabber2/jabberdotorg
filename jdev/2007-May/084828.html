Today I set up static DNS zones for the first two interop servers:

- ejabberd.interop.xmpp.org
- openfire.interop.xmpp.org

It seems that both servers are listening on port 5222. I know that the
openfire server has in-band registration turned off. I think that all
servers must do that so we have controlled access to this network.

I have also finalized the Confidentiality Agreement:

http://www.xmpp.org/xsf/docs/interop-confidentiality-agreement.shtml

Details about how to get involved are temporarily parked here:

http://wiki.jabber.org/index.php/InteropTestingNetwork

Once we set up http://interop.xmpp.org/ we will move the information
over to that domain.

Ping me with questions and let's start testing!
