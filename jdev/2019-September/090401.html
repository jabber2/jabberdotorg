Conversations sends chat markers to the entire room in order to be able
to display to everyone who's seen which message, not just to the sender
of the message (at least for private groupchat).  So it probably just
doesn't have the necessary code to support your model, i.e. to associate
a chat marker received via PM to a groupchat message.
