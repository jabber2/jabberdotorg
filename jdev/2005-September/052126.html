Hi,

I'm very pleased to announce the first release of XMPP4R : XMPP4R 0.1.

XMPP4R is a XMPP/Jabber library for Ruby. It provides all the necessary
backend to write scripts or applications acting as Jabber clients or
components.

While this first release is not feature-complete yet (SASL, TLS and SSL
support are still missing for example), it already allows to write most
of the applications you will need.

XMPP4R is fully documented using rdoc, and comes with a lot of examples.

XMPP4R is released under the Ruby license, which is GPL compatible.

Related URLs :

XMPP4R 0.1 : http://download.gna.org/xmpp4r/xmpp4r-0.1.tgz
