    Well, thanks for the support and contributions, I have been
receiving a lot of emails for jabberfs ...

    As everybody mentioned, perhaps I should release the code, even if
it is still very experimental and unstable ... I have been thinking
about this, and since people here are mostly developers too, I believe
you know how I feel and will be patient with an unstable code ( you
are the first to put your hands in it ! ) ... so, here it is
   http://www.lsc.ic.unicamp.br/~samuel.goto/jabberfs/jabberfs.html

   I tried to use autotools to distribute the code, and this is my
first release with autotools ... if you find any problems compiling
the code, let me know ... you will need fuse (fuse.sf.net) instalated.

   Well, hope you enjoy, send me some bug (there will be, don't worry
if you find one ... ) and feature request,
