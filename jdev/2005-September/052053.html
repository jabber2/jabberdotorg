Hi folks,

I'm happy to announce the immediate availability of gloox 0.6.

With this release gloox is fully compliant to XMPP Core and IM (with two 
exceptions -- see below). Additionally, an experimental implementation of 
JEP-0012 (Last Activity) has been added.

http://camaya.net/gloox

Not supported but required by XMPP IM:
- End-to-end object encryption as defined in RFC 3923
- handling of addresses that are encoded as "im:" URIs

cheers,
Jakob
