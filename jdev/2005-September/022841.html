We are pleased to announce a new release of the Open Source Jive Messenger 
server.

The new release includes TLS (c2s), SASL (c2s) and PAM authentication 
support. The complete change log can be found at 
http://www.jivesoftware.org/builds/messenger/docs/2.3.0.alpha1/changelog.html. 
You can download the server from http://www.jivesoftware.org/messenger.

This is an alpha release not because the code is in alpha state but because 
we will continue adding new features for the final version. Please post beta 
feedback in the support forum.

Enjoy!
