Hi,

I implemented a Jabber client that watches the web browser and automatically logs in to chat rooms while browsing. There is a mapping from (Page-)URLs to (Room-)JIDs. Users in chat rooms appear as avatars on the web page. It is backward compatible to other clients and conf components (Groupchat and MUC). 

It's an open source project on Jabberstudio. I would like to publish the protocol extensions and the URL-to-JID mapping as a JEP. Please have a look at http://www.lluna.de/docs/jeps/vp/jep-vpprotocol.xml

Website: http://www.lluna.de/
Proposed JEP: http://www.lluna.de/docs/jeps/vp/jep-vpprotocol.xml
