LLuna, a Co-Browsing and Chat on Web pages Jabber client, is now in beta test for Version 2.5. 

LLuna is an open source project for Virtual Presence based on the Jabber protocol. 

New features: 
- co-browsing, web-guide, 
- annotations on the web page, 
- temporary buddies from virtual presence.

Website:  http://www.lluna.de/
Download: http://www.lluna.de/download.php
Source:   http://www.jabberstudio.org/cgi-bin/viewcvs.cgi/lluna2/
