The CVS version of jabber.el (http://emacs-jabber.sourceforge.net) now
supports most of JEP-0050, Ad-Hoc Commands, both as a server and as a
client.  It uses this to enable setting presence remotely, e.g. if you
have forgotten to set a distant client to "away" status and low
priority before leaving.  Access to this feature is granted only to
entities with the same bare JID.

I want to test my implementation against others, but I haven't found
any.  Is that simply the state of the world, or have I missed any
program?
