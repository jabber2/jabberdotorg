This new version has a slough of new features, including most of XMPP
-core and -im.  TLS, SASL, S2S, XMPP error-exception translation,
stringprep prohibit and quite a few other things are included.  As
always, our library runs on the Standard .NET Framework, the Compact
Framework, and the Mono framework.

>From the release notes:
+ Support for the following SASL authentication mechanisms
	+ DIGEST-MD5
	+ ANONYMOUS
	+ PLAIN
	+ EXTERNAL
+ Support for the XMPP STARTTLS protocol
+ Support for XMPP resource binding
+ Support for XMPP session initiation
+ Stream Features support
+ ClientAuthenticator class
+ Session.Login method
+ Peer to peer file transfer
+ Support for XMPP exceptions
+ Completely re-written trace library
+ New Performance Counters
+ Updated TLS security library
+ Greatly increased packet de/serialization speed
+ Support for payloads on every packet
+ Enhanced JabberID validation on Desktop Framework using StringPrep
+ XMPPDnsResolver for SRV record lookup for both client and server
records
+ WConvert class for converting between polymorphic packet types
+ Dialback support
+ OnXMLSent/Received events added to Session
+ Support for current Mono version (0.30)
+ TraceMessageAdapter for serializing trace messages into MessagePacket
objects
+ GenericIQPacket for easier custom IQ implementations
