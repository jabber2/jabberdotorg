On Tue, 6 Jan 2004 23:12, Nauman Khan wrote:
> Hi All,
>
> Is "Thread TAG in Message Syntax" the only way to conduct multiuser chat ?.
> Is there anyways we can specify more than one JIDs in a message ?

No.  But there are some ways you can go.

1. Address multiple packets, one per recipient.  This is what clients will do 
if they support the entry of more than one JID in the To field.
2. Groupchat / MUC.  This is for general multi-user chat scenarios.
3. Publish / Subscribe.  This is somewhat analogous to a mailing list.
