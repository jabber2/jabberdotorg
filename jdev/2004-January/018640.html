> We have created a KNOPPIX CD including a Jabber Server.
> 
> KNOPPIX is a Linux Live System which boots and runs from CD,
> with out using (doing changes to) your Hard disk. All you need
> is an i386 architecture PC with at least 256 MB of Ram (I would
> recommend 512 MB) to run a complete Linux Server (including Jabber
> Server).
> 
> In case the PC or Laptop runs inside a LAN supporting DHCP,
> at boot time the Jabber Server is configured to run on the IP assigned
> by DHCP, and so it is accessible from any PC inside the LAN.
> 
> We use the Live CD for Marketing and Presentation of our HTML based
> 'Open Architecture Community System' (http://openacs.org) which is build
> on top of Aolserver (TM) (www.aolserver.com) with Oracle or Postgres as
> RDBMS backend.
> 
> We work on a integration of Jabber as IM Solution, witch already runs
> on Oracle based Platforms (and our Custermors are very satisfied with
> it. Thanks to the Jabber Members. ).
> 
> The Live CD uses Postgres, and the porting of the Jabber Package for
> the Postgres based OpenACS isn't sure to be finished before the Linux
> World Expo.
> But you sure be able to create a new  user accounts at the
> Jabber Server and automaticly logging in to it, with the (Java based)
> Greenthumb Client, by Clicking on a Link provided by our Community
> System. The Community System  will be available through any Web Browser
> from inside a LAN, as well.
> 
> This makes it quite easy to present Jabber to your Customers,or let
> them try out Jabber inside their Company by giving them a Live CD.
> All data (user accounts , contacts etc.) will be raised once the
> Linux System running from the Live CD is shut down.
>
