Am Mon, den 12.01.2004 schrieb Jim Parslow um 13:29:
> Using the jabber:component:accept method
> 
> In the jabber.xml I have <archive><service>component</service></archive> and 
> it seems to send 2 routed packets through, any idea why?


If user A sends a message to user B, one packet is from A to jabber and
one is from jabber to A. There is a patch for jabberd-1.4.2 around
anywhere which adds an direction to the packets so that you can
distinguish which is which. I think a found the link to this patch in
msglog's README.
