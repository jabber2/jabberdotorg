FYI, I've worked over the last few weeks to update the XML schemas for 
all approved protocols (Final, Draft, and Active JEPs), and to define 
XML schemas for older JEPs that did not contain them. Each approved JEP
now specifies canonical URLs for the relevant schemas. I have tried to
make the schemas as accurate as possible, including proper ordering of 
attributes and specification of default values in order to assist
developers in implementing XML canonicalization if desired. Naturally, 
if you find any errors, please let me know.

I will also find an appropriate place for the schemas that are defined
in the XMPP Internet-Drafts -- so far I've placed online only the XML
schemas that are defined in JEPs.

Thanks.
