> Does anyone know the method of sending a message to multiple ppl ? like in
> Conference chat on MSN ? Can message type 'groupchat' help in it ? I want
> to say that if i send a message with type 'groupchat' then is there any
> method i can specify more than one recipients and then Server is able to

Funny, someone asked the exact same question less than 48 hours ago, and I 
gave a much more verbose answer last time.

JEP-00045 Multi-User Chat: http://www.jabber.org/jeps/jep-0045.html

MUC is an extension on the old groupchat protocol.  The MSN transport seems to 
use a groupchat protocol to implement MSN's multi-user chats but MUC is 
considered current.
