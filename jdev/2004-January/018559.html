Of course it is, I was trying to make a point ;)

Main point is that the current Jabber network is very vulnerable to 
things like flooding, (d)dos attacks, spam, etc. Very understandable 
because of the enormous trust that all jabber users have in each other 
but this will prove to be a (severe) problem in the near future. I 
really hope that it won't have to come to the shutdown of some public 
servers before everyone realises how vulnerable we really are.

Web-based registrations are one way to solve a part of the problem. 
Furthermore it would be nice if everyone could take a look at this page:

http://www.jabber.org/wiki/index.php/spim
