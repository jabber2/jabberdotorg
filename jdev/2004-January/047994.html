Hi all,

Many folks have been asking about how to get an http polling connection with 
Jabber, and now there is a new option: Tonneru, by Akito Nozaki.  It's 
written in C++ and has two parts: a cgi and a daemon.  It's easy to set up 
and works with any Jabber server.

  http://tonneru.tanoshi.net/
