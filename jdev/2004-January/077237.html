Hi,

I am just now (actually today) developing a Co-Browsing for Jabber. Maybe my code fits.
What do you exactly mean by Co-Browsing?

I can say what I do:
- A person (called guide) has a browser window open.
- The guide clicks a button attached to the window to start a cobrowsing session with this specific browser window.
- Other persons (follower) have a browser window open.
- The follower clicks a button attached to the window to start a cobrowsing with this specific browser session.
- There may be multiple followers.
- The guide navigates the browser window to a URL (by entering the URL or bookmark or click on a link).
- The browser windows of all followers navigate to the same page.
- This is not a shared screen application. Each client does the HTTP and fetched with its own client state with respect to access rights, cookie, session, if any. Only the URL tree of the guide's browser is transmitted to the followers.
