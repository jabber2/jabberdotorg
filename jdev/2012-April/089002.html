Indeed. There's nothing to stop a server providing access to an
archive via HTTP, either in a machine or human-readable form. We just
don't have any specification for it.

query an archive over XMPP for alternative access methods (e.g. to get
a HTTP URL), and even specifying what format the request and response
should be.
