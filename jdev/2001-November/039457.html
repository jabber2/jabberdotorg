JabberFoX 0.4 just has been released! You can find it at 
http://jabberfox.sourceforge.net/download.php


JabberFoX is a Jabber client for Mac OS X. It is written in 
Objective-C using Apple's Cocoa API and tries to make full use of 
Aqua, Apple's cool new user interface system. It aims at implementing 
all of the standard jabber features including chat, group chat, 
messages, the roster (buddylist), and agents. Eventually we hope to 
incorporate all of the features of the Jabber protocol, including 
browsing, strong encryption and the new conferencing protocol.

JabberFoX is currently being maintained by a small group of volunteers.


News in this version:
* Localized to French and German
* Message In/Out dialogs revamped
* Added support for styled messages!
* Preference dialog now is similar to the one in Mail.app
* EventManager - have it blink when a new message arrives, or bounce 
the dock icon for each chat message
* Added tooltips in the roster and some other places
* Added Dock Menu in favor of the global menu
* GroupChat join is now handled by a separate dialog
* In the GroupChat join dialog, the "Add To Favorites" button now is 
implemented
* Removed "Add Contact" button from roster and made it a menu command
* Minimal size of roster reduced so you can save some space
* Simple "dragging" of roster items now possible (but no "drop" in 
the roster), useful to insert a JID somewhere
* As usual, tons of bug fixes and minor enhancements


Do you want to learn about future releases of JabberFoX? Then 
subscribe to our announcmente mailing list here: 
http://lists.sourceforge.net/mailman/listinfo/jabberfox-announce
