Hello,

  In order to use the JCF irc-t, you have to use the 'Browse Conference'
option, select the transport on the list, and then either create the
room or enter an existing room. The 'IRC' option under groupchat is only
for the very old version of irc-t, which stopped being supported with
the CVS version. 

  I'm currently trying to finish the patch for the CVS version so that
it can be used by clients supporting the conference protocol (not
gabber, but winjab and rival) This is aiming to be able to work
alongside the JCF irc-t .. in fact that is how I have it set up on my
own server.

Regards,

  David
