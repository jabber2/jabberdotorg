Its still a bit rough & ready (I've only tried Solaris & Linux), you
also need new autoconf/automake/libtool:

autoconf (GNU Autoconf) 2.52
automake (GNU automake) 1.5
ltmain.sh (GNU libtool) 1.4.2 (1.922.2.53 2001/09/11 03:18:52)

Apply the patch (against current code in CVS), run autogen.sh,
configure & make.

http://www.alexandalex.com/~alexk/jabber.diff

I haven't rolled in the checks which the existing configure script had
yet, nor SSL enable/disable, but its there if anyone's interested in
it as is.

The dist stuff doesn't work yet (mostly it doesn't pick up all the
files it should), also install puts things in different places than
the jabber.xml config file has so you need to do a bit of changing by
hand.
