The O'Reilly Open Source Convention is currently accepting proposals for
talks and tutorials (deadline this Friday). Several Jabber folks have
already submitted proposals but of course it's an open process and more
are welcome. :) You can find out more here:

http://conferences.oreillynet.com/os2003/
http://conferences.oreillynet.com/cs/os2003/create/e_sess

They have tracks on Perl, Python, PHP, Apache, XML, Ruby, etc. If you're
working on those areas and are interested in speaking, send in a proposal!

BTW, the JSF is working hard to make sure that there is a Jabber booth and
a fairly major Jabber presence at OSCON this year. If you have questions
or just want to bounce around some ideas, fire away.
