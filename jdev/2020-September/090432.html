Hello everyone,
Tigase team is pleased to announce new releases of BeagleIM 4.0 and 
SiskinIM 6.0

More information can be found in blog post:
https://tigase.net/beagleim-4.0-and-siskin-6.0-released/ and in the 
respective github repositories: 
https://github.com/tigase/beagle-im/releases/tag/4.0 and 
https://github.com/tigase/siskin-im/releases/tag/6.0


Most important features and changes include:
* support for XEP-0369: Mediated Information eXchange (MIX)
* improved VoIP connectivity by adding XEP-0353: Jingle Message 
Initiation (now compatible with Jingle Message Initiation used by 
Conversations)
*  support for XEP-0308: Last Message Correction
* support for XEP-0424: Message Retraction
* support for quick replies - quoting messages to improve conversation flow
* support for XEP-0158: CAPTCHA Forms: Extended In-Band Registration
* redesigned channel joining UI
* optimized resource usage during the establishment of VoIP calls and 
improved quality of VoIP calls
* improved Markdown support
* improved message synchronization speed, reliability and add support 
for MAM version 2 (XEP-0313: Message Archive Management)
* started using MAM for MUC history synchronization on rejoin (if possible)
* improved support for CAPS
* new chats list view with a single section for 1-1 chats and group chats

We welcome all feedback - https://github.com/tigase/beagle-im/issues and 
https://github.com/tigase/siskin-im/issues
