> have a custom tag that I place within the status message to pass this
> information.
> The last rev of the jabber server I tried (shortly after the protocol
> change) no longer allowed this functionality. Is this still the case?

  The status packet can be extended with the use of additional tags using a 
custom namespace.  Aka, <mynewtag xmlns="jabber:mycustomnamespace>.  I BELIEVE 
this is what is being spoken about in the current protocol documentation when 
it talks about the <x> tag..
