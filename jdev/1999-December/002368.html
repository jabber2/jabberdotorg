On Tue, 21 Dec 1999, Todd Kirby wrote:

> I tried to install Jabber on two different machines
> 
> Redhat Linux (Hedwig) i386 on a compaq
> LinuxPPC 1999 Q3 (based on Redhat) ppc on a Mac
> 
> The installs seemed to go well but, when I try to
> start etherxd (or jserver) I get the same error on
> both machines:
> 
> etherxd: error in loading shared libraries:
> libjabber.so.0: cannot open shard object file: No such
> file or directory
> 
> The shared libs do exist in /usr/local/lib/

You should use the Red Hat packages from
      http://lm.lcs.mit.edu/~cananian/Jabber
which install the libraries in /usr/lib.  /usr/local/lib is not in the
default library path for Red Hat boxen --- look into /etc/ld.so.conf and
see for yourself.  If you configure with
  ./configure --prefix=/usr
things will work better for you.
