To help get 'cha started with the 0.8.1 release of jabber as quickly as
possible, I've built Red Hat RPMs for the 3 pieces of the jabber server
(libjabber, etherx, jserver) and put the binary and source packages at
  http://lm.lcs.mit.edu/~cananian/Jabber
I'll be trying to keep these up to date, and the server RPMs will probably
be joined shortly by packages for libjnix and cabber, which seem to be
the most robust of the jabber clients that I've been able to build.
Share & enjoy...
