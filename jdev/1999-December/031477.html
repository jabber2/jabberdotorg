WAP handsets (Nokia 7110's) are available in the UK now (from Orange),
they are dual band (GSM/PCS i.e. 900/1800MHz).

The current WAP spec is v1.1 (from www.wapforum.org), v1.2 is going to
be ratified next week in Sydney, this will have extra features like
client side certificates and other security enhancements.

Most phones will only support microbrowsing (i.e. WML/WMLScript). So
any IMPP type service will need a thick client running on a server
and an http nterface on to it.

In future as WAP terminals get better (in terms of CPU/Mem) then you'll
be able to run a real WAP client in the terminal which runs a thin
(say) IMPP client which in turn talks to a IMPP server directly (or
via middleware on a server).

The Ericsson MC218 should be available (which is a Psion 5mx, but
rebadged with a WAP stack and microbrowser on it). This should be
able to run applications directly.
