[Crossposted from java-dev]
The Jabber Extensible Client Interface for Java is now available for use and
properly documented in the "java-webb" module on CVS. This is the client
interface being used by Webb in its own (upcoming) Jabber applets, that
works with Jabber server 0.8/0.8.1.

The doc/java-webb.html file contains extensive documentation on the use and
extension of the JECI. Feedback is encouraged, as I expect some other
developers here will be making use of these docs eventually, and I want them
to be clear.

Happy hacking...
