Rene,

I'm afraid I won't have time this myself at the moment, but wanted to 
let you know that with this release, building JabberD under Cygwin works 
like any other *nix platform now:

	./configure
	make

or
	./configure --enable-ssl
	make

(assuming you have appropriate Cygwin packages installed of course for 
things like OpenSSL, etc.)

So if you're in a hurry, just install Cygwin, download GNU Pth and 
JabberD 1.4.3, and build like any Linux user might.  Only exception is 
the new jadc2s (see other post).
