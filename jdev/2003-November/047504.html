I'm not deeply familiar with WASTE yet, but it strikes me that the same
functionality could be developed using various Jabber protocols. IM and
chat come out of the box, and we also have file transfer and related
features using these protocols:

http://www.jabber.org/jeps/jep-0065.html
http://www.jabber.org/jeps/jep-0095.html
http://www.jabber.org/jeps/jep-0096.html
http://www.jabber.org/jeps/jep-0047.html
http://www.jabber.org/jeps/jep-0105.html

File discovery could be done using service discovery or an extension
thereto:

http://www.jabber.org/jeps/jep-0030.html

The private networks could be done by setting up dedicated servers that
are not connected to the rest of the network.

Encryption could be done using one of these:

http://www.jabber.org/jeps/jep-0027.html
http://www.jabber.org/jeps/jep-0116.html

We're also talking with folks about native client auth using Kerberos.

Just a thought....
