Jabberd 1.4.3 has been released. Jabberd is an open-source implementation
of a Jabber server software.

The source and documentation is available at
http://jabberd.jabberstudio.org/1.4/
For a list of changes, see
http://jabberd.jabberstudio.org/1.4/release-1.4.3.shtml

md5sum  a3e964d6fa07b5d850302ae0512f94c6  jabberd-1.4.3.tar.gz


Additionally, jadc2s 0.9.0 has been released. jadc2s is a Jabber server
component that handles c2s (client-to-server) connections. It can be used
as a faster replacement for jabberd's pthsock_client component.

Source and documentation is available at
http://jabberd.jabberstudio.org/1.4/doc/jadc2s
