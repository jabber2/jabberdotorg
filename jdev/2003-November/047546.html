Building from source is easy - I avoided it, but all I did was to 
install gcc,gc++,,gcc source,GNU make etc and found it very easy. All 
you do is (to compile jabberd for example):

$ ./configure --options
$ make

Wait for a while (or some more while on my slow PC) and it is done. Some 
apps need a 'make test', 'make install', 'make menuconfig' etc.

In this way, I have set up jabberd-1.4.3 under cygwin.
