Hi all!
Tigase team is pleased to announce 8.1.0 release of Tigase XMPP Server -
fast and powerful server capable of handling millions of users.

More information can be found in blog post:
https://tigase.net/tigase-xmpp-server-8.1.0-ga/

Most important features and changes include:
* SASL-EXTERNAL for s2s connection
* support for XEPs:
   - XEP-0398: User Avatar to vCard-Based Avatars Conversion
   - XEP-0156: Discovering Alternative XMPP Connection Methods - Tigase 
already supported handling DNS queries and standardised our webservice 
to XEP-0156
   - XEP-0410: MUC Self-Ping (Schrödinger’s Chat)
   - XEP-0153: vCard-Based Avatars - added support for setting vCard 
avatar for MUC rooms
   - XEP-0411: Bookmarks Conversion
   - XEP-0157: Contact Addresses for XMPP Services (server-995) that can 
be configured on per VHost basis

* improved handling of multiple domains configuration - it's now 
possible to configure set of defaults for all domains and also easily 
extend domain configuration with custom extensions
* push notifications can now be encrypted
* it's possible to register in MUC to receive push notifications on 
mobile devices (though we are going to include MIX in next release, 
which should alleviate majority of problems with groupchat on mobile 
devices)
* improved hardened-mode

Release can be downloaded from
https://github.com/tigase/tigase-server/releases/tag/tigase-server-8.1.0

We welcome all feedback - https://github.com/tigase/tigase-server/issues

Wojciech Kapcia
