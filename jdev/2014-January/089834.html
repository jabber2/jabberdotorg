Hi all,

Whilst helping Simon Tennant with building the Buddycloud XEP I've put
together a small XEP building tool/helper which can build a XEP whilst
editing and helps split the document down into smaller sections whilst
doing so.

Its available on github here https://github.com/lloydwatkin/xep-builder

A slightly older version is being used to build the Buddycloud XEP and
being automatically built and deployed to
http://buddycloud.com/buddycloud-xep by Travis (from here on commit
https://github.com/buddycloud/buddycloud-xep).

It probably doesn't do everything required properly but if people find it
useful I'm happy to spend some more time on it.

Cheers, Lloyd.
