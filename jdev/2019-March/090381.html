Hi,

Tigase team is pleased to announce 8.0.0 release of Tigase XMPP Server -
fast and powerful server capable of handling millions of users.

More information can be found in blog post:
https://tigase.net/blog-entry/tigase-xmpp-server-800-general-availability-released

Most important features and changes include:
* significant internal rewrite (Tigase Kernel - implementation of IoC)
* new configuration format
* support for XEPs:
  - XEP-0215 External Service Discovery
  - XEP-0305 Quickstart
  - XEP-0313 Message Archive Management (MAM)
  - XEP-0357 Push Notifications
  - XEP-0363 HTTP File Upload
* CAPTCHA system in XEP-0077 In band registration
* anti-spam mechanisms
* protection against brute-force attacks
* passwords are now hashed by default
* database schema versioning

Tigase Server can be downloaded from
https://tigase.tech/projects/tigase-server/files

Upgrading from previous version is described in details in documentation
(https://docs.tigase.net/tigase-server/8.0.0/Administration_Guide/html/#_upgrading_to_v8_0_0_from_v7_1_0)
and involves upgrading database schema.

We welcome all feedback - join discussion in our forums
https://tigase.tech/projects/tigase-server/boards

Tigase Team
