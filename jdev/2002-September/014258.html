What
>other JIM like clients for windows users is worth giving a crack at that
>are free and supported?

Exodus is a free Windows client that is pretty popular. It usually is among 
the first clients to have a lot of the new features. Plus, it has support 
for SSL, in case security is an issue.

You should be able to get a lot of support on this list as well as on the 
jadmin list.

There are a lot more clients (Check for clients on www.jabbercentral.org) on 
Windows, but I haven't used any as much.

Hope this helps,
Ragavan

PS: Disney's GO Messenger is another Windows based client that you could 
try. I think you can download it from http://im.go.com
