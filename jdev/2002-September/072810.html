Some preliminary network sniffing by some friends of mine suggests
that Apple's iChat client, which uses AOL's OSCAR protocol to
communicate with AOL's IM servers, uses Jabber when communicating with
other iChat clients when it "discovers" them via Rendevous/Zeroconf.

This is _very_ interesting news to me, for despite the work of the
jabberfox and tvjab folks, the state of MacOS client support for
jabber is still notoriously shaky.  Does anybody from the Jabber
foundation have any comment on this?
