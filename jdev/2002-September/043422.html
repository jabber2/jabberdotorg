Hi!

I just read the "groupchat 1.0" protocol jep-0045. There are features
mentioned like kick and ban. As far as I know this is not supported by
the conference component at the moment.

The jabber community might be interested that we are currently working
on a server component for moderated groupchat. It's working as proxy in
front of the original conference component filtering the inbound and
outbound traffic. It's written in perl using Net::Jabber.

For the big picture just have a look at http://jabber.bluehands.de
