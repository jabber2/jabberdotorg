On Thu, 19 Sep 2002, dave zonstein wrote:

> In the last December a message from Thomas Muldovney mentioned that server
> side buddy list support will be added to the transport.
> Was this added already, or will this be added?

Currently neither AIM-t nor ICQv7-t support automatic buddy list import
from AOL servers. I don't know of anyone working on it either but for
ICQv7-t such a feature would be more likely and easier to implement I
think.

> Does icqv7-t support it and is it a transport to be used instead of the
> AIM-T for ICQ?

As mentioned on aim-transport.jabberstudio.org ICQv7-t is better suited
for ICQ than AIM-t is (better SMS support, direct connections, getting
away messages).
