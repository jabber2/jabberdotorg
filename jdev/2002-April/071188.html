The MSN gateway isn't running on jabber.org -- try a server that has it
running (http://www.jabberview.com/ has a list of servers).

If you're trying to write a client, why not run a test server on your own
machine instead of connecting to jabber.com or jabber.org? To test any
server, use telnet instead of your own client (whcih may have problems).
Here is text that can help you:

http://www.saint-andre.com/jabber/telnet.txt

In general it's a good idea to use the telnet interface as a way to learn
the XML API.
