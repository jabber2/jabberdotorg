Marc and myself are please to announce that we have an Alpha version of
Yabber, our Win32 Jabber client, available for download on our website.

Yabber is designed to be a solid Win32 client for both new users and power
users.  Special detail has been given to the user interface (which has been
based on my design tutorial document
http://www.aurora.gen.nz/jabber_design).  The look and feel is modelled
loosely on ICQ to give new users some familiar ground when moving to Jabber.

Features include:  dual message type support, file transfer, original sounds
& graphics, docking, auto-hide, auto-connect, transport support etc...

While still a work-in-progress and largely untested, we hope Yabber is
stable enough to use day-to-day.  We would appreciated feedback (both bugs
and feature requests).  Feedback details are on the website (email or logged
directly to Jabber Studio)

Grab a copy from http://www.yabber.org  or just stop by and check out the
screenshots.

We have heaps of ideas on how to extend Yabber, so keep checking the website
for future versions if you are interested.

Michael.
