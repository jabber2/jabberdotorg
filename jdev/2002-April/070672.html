OK, more fun with Jabber: reatmon and temas (the JabberStudio guys) have
added a feature that enables you to receive a Jabber message whenever
files in a CVS repository or a specific subdirectory have been updated.
I'm already using it for JEPs and it's really handy. All you need to do is
add a .cvslog file to the appropriate repository or directory and include
a line such as 'jabber://stpeter@jabber.org', then wait for the Jabber
messages to arrive. More information is here:

   http://www.jabberstudio.org/news/view.php?id=19

Great work, guys!
