Hi,
  Presently only jabber.com server supports http proxy (As described by JEP-25 http://www.jabber.org/jeps/jep-0025.html) You can connect to jabber.com using Exodus(Only exodus supports HTTP polling methords among the stand alone clients) or JabberINC's web client. There are presently no implementations of this on the open source server side. 
  If you are having problems logging in using Exodus to Jabber.com, check your url in the proxy settings window. The link should be: http://webim.jabber.com/wc12/webclient/. You are free to give any # in the polling time box (Give larger number if your Net connection is bad. Voila, You break through you firewall and land up in the jabber network.
