Hi every body !

I have some misunderstandings on XEP-0060 PubSub.

What I understand :

A node can be :
- persistent or transient
- notification or payload

When someone publish to a node (for example persistent/payload), event
notifications to subscribers can be only-notification or payload-included,
depending on pubsub#deliver_payloads value.
If pubsub#deliver_payload if false, payload are not sent but can be
retrieved by protocol "Retrieve items from a node ( chapter 6.4 in XEP ).

BUT, if what I understand is right, I don't understand followings :

- why a persistent / notification node can contain a payload ? (table 5 in
XEP)
- how to set notification or payload configuration. There is no field in
pubsub#node_config to set it.

Thanks of lot to give me the right understanding of this XEP.
