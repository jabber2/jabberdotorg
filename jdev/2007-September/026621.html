On Thu, Sep 13, 2007 at 11:17:10PM -0500, jdev at eriko.mobi wrote:
> Hi, all,
> 
> I'm trying to write a simple Perl script to log in, get a list of
> subscribed JIDs that are online, and then log off. 

I don't know about Net::XMPP, but I just wrote a simple script that
does exactly that for Net::XMPP2, you maybe want to try it, you can
find the script in the git repository for Net::XMPP2:

   http://www.ta-sa.org/files/git/gitweb.cgi?p=Net-XMPP2.git;a=blob;f=samples/retrieve_roster;hb=HEAD

It should work with a recent CPAN release of Net::XMPP2 or the version
in the repository. See also http://www.ta-sa.org/net_xmpp2

You start the script like this:

   # ./retrieve_roster testjid at server.tld yoursecretpassword

It should then just connect and print the stuff to stdout or
timeout if something gets stuck. Or print an error if something
unexpected happened.
