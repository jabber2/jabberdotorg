David Conner wrote:

> Actually, this is part of what confused me.
> At http://protocol.jabber.org/docs/identity.html,
> resource is defined as "the optional third
> identifier," which implies that it's optional. :)

Don't be fooled - it is required for logging in (all users must have
resources) but not required for sending a message - if you do not
specify one it goes to whichever user is the 'default', where default is
chosen by whichever resource has the highest 'priority' set in its
presence.
