Hello,

currently mod_version is sending xml message with the current servers
version to update.japper.org. I asume this feature is used to get a admin
message back if there is a new version of the server out there.

On the other hand this is a dangerous spyware feature. Although it is easy
to remove the mod_version from the xml config, I think it would be better to
disable it at default, or be very clear about this feature in the docu.

What is done with the Data on update.jabber.org? Are there statistics? Will
it be logged?

I suggest we open up the access to those statistics. Then we can ask every
admin for help and use mod_version... kind of Jabber Counter Project.
