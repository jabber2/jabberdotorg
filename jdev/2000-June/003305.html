No, we're still there. It's old because it hasn't needed to change
much.

There is a slightly modified version in CVS, which does have a major
bug fixed. If you can, you'll want to grab it from the CVS server
instead of download.jabber.org.

You mention in a later message that you're using windows. That ought
to make no difference at all. What would make a difference would be if
you were using a version of Python older that 1.5.2. If so, you need
to move up to 1.5.2 or newer.

taking so long to get back to you.
