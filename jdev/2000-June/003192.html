There shouldn't be a cvs version of java-mass at all - it has been moved
to http://jabberbeans.sourceforge.net. The code in java-mass would be
rather behind now - I've done a lot of work in the last 3-4 weeks.

java-webb is a separate API for interfacing with a Jabber server, done by
erbo for Webb. It is located in the jabber.org CVS.

There aren't any applets currently published, but there will probably be
many published soon. The more the better!
