In case you missed:
We published a Windows sample app for libjingle. It includes the media 
engine for Windows, which is missing in libjingle. The media engine has 
been made GPL for this purpose. Any Win32 client developer can use the 
MyJingle code (http://www.bluehands.de/software/beat/myjingle/) to add 
Google Talk support to their clients.

We built this sample app while doing the research on VoIP support for LLuna.
