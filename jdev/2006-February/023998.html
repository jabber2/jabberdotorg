Hello all,

We're pleased to announce the 1.1 release of the Spark instant messaging
client. Spark is a cross-platform XMPP client with a rich feature set
and professional look. This happens to be quite a big release. :) Major
changes include:

 * A Linux/Unix version. Mac support has been upgraded significantly,
and of course Windows is available as well.
 * Support for Sparkplugs (plugins). Read more at:
   http://www.jivesoftware.org/articles/sparkplug_day.jsp
 * Greatly improved file transfer. Read more at:
   http://www.jivesoftware.org/articles/filetransfer.jsp
 * Numerous bug fixes and improvements.

A full changelog is available at:

http://www.jivesoftware.org/builds/spark/docs/latest/changelog.html

and full Spark information and downloads are at:

http://www.jivesoftware.org/spark/

A new release of Spark Manager is also available and support Linux.
Spark Manager offers server-based control of Spark installations and can
be found at:
http://www.jivesoftware.com/products/sparkmanager/index.jsp

Regards,
Matt
