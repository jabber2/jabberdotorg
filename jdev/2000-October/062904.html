I also posted an announcement at:
	http://www.jabber.org/article/42.html

The 1.2 Jabber Server Release Candidate is available at:
	http://download.jabber.org/dists/1.2/

It is currently running the jabber.org server (and has been in various
states for the past few weeks).

This is a test release, it is feature complete and should be fully
functional and has all the known bugs fixed. Please take this chance to
test it before the upcoming final release.

We're also taking this time before release to work on packaging, and are
looking for assistance in building debs, rpms, solaris packages, and any
other common package format. We're also looking for assistance in testing
on platforms other than linux, [free|open]bsd, and solaris.

Thanks,
