Hello Bernd,

  The irc_gateway is not currently running on jabber.org.. Jabber.org was recently switched over to jabberd, and the irc_gateway is not an internal part of "jsm" anymore.. 

  The good news is.. irc_gateway was rewritten to work as a jabberd component, and is a part of the irc_jabber module.  It isn't running currently on Jabber.org, It needs to be updated for a few last minute protocol changes that have happened since 1.1.2.. (internal protocol changes that is).

  I will definitly put that on my TODO list, and get that updated, and back up running as soon as I find the time.. glad you like the cool tool =]

  as a technical note,  just jabber.org:6667 will work.. irc.jabber.org technically is the host name for irc-transport, not irc_gateway.. but externally, DNS for jabber.org resolves *.jabber.org to the IP address of jabber.org, which is why irc.jabber.org:6667 works.  but internally to jabber, they mean very differently things... irc.jabber.org is the jabber->IRC groupchat gateway, whereas irc_gateway allows you to log into jabber using the IRC protocol.
