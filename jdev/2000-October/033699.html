There was a PHP client called Phabber but it never really got beyond the alpha
stage since the developer didn't have time to keep working on it. There is some
HTTP work going on at jabber.org and jabber.com, but it's not quite ready for
prime time yet. We're hoping to see some fairly robust HTTP-based client-server
interaction in version 1.2, but I'm not sure when exactly this functionality
will be finished (to some extent it's not a core part of the server, at least
compared to things like the basic architecture and the scalability efforts going
on).

Hope this helps. Let me know if you have any further questions.
