> Is there a maximum size of a Jabber ID?

yes and no...
there are 3 parts to a jid, 
username -- limited to 255 *valid* characters as per the JPG
hostname -- limited the same as a FQDN hostname
resource -- no lenght or valid character restriction

> If its yes to any of these, does the server 
> enforce these maximum lengths?

yes, the server enforces these restrictions, technically, the enforcing is done inside libjabber's jid_new() command, and will return NULL on a bad jid.
