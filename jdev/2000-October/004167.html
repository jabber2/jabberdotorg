I can answer the question about the Jabber User Directory.  That agent
is not a place where you can send messages from or anything like what
your roster is.  It is simply a database that you can search for people
to add into your roster one by one.  So you still have to add them in
like that, but if you aren't sure of their Jabber ID you can use JUD to
look for them based off name or email.  Also, you have to register
manually with JUD.
