Quoting Nickolai Kussovski <nike at webmessenger.bg>:
> Hi,
> does anybody know which is the user database with Jabber 0.9 and how to add
> users to it?

  In the case of the standard installation, the file xdb is used.  If you look 
in your jspool directory, most likely /usr/local/var/jspool, you will notice 
files by the name of username.xml.  This directory is the 'user database'.  
Each user has one file, which contains all data for that user, including 
roster, offline messages, etc..

  Most of the time, this file would be created on a new user registration.  If 
you wish to manually create a user, simply copy one of the existing files, and 
change the name/password information.
