I'm looking at packaging at least a couple of transports. What sort of
automatic configuration do you want? DYM setting the hostname for the
server or more complex stuff?

For the likes of installing a transport doing automatic configuration is
tricky - it requires adding things to the server configuration file. I'm
not really that keen on packages munging text config files. Perhaps a
patch and then a debconf query to ask if you want to patch the main
server file to include the transport might work.
