Joogle Agent is a Jabber agent that can perform a Google search for
you through a Jabber IM. Any messages sent to it (currently running at
joogle at jabber.com) will be treated as a search string for Google.
Results are returned in a series of messages.

This has absolutely no practical real-life use, other than to brush up
my Java =). At the very least I suppose it provides some simple
example code for using JabberBeans.
