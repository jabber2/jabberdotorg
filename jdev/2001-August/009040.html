just to tell everyone, the Profiles JIG is now up and roaring along.

Summary: Profiles is all about personal information storage. Better than
the current system because it allows for granular access, flexible read
and write permissions, and distributed remote storage, among many other
features.

Details: http://foundation.jabber.org/jeps/jep-0006.html and
http://www.theoretic.com/profiles

I encourage everyone who is interested in the next generation of
information storage and retrieval in jabber to participate in the
Profiles JIG. the mailing list is at:
http://mailman.jabber.org/listinfo/profiles-jig

we are now discussing the read/write permissions, and plan to have a
rough spec ready in a week. this first spec will not be the final
release. far from it, it will only be a 'patch' to the current system,
to get something ready for the 1.4.2 server that is about to be
released.
