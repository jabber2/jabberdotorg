Dillip, if you want to use the dynamic version then you need to install
expat using:

$ make
# su 
# make install

You would need to do this on EACH test machine that runs a jabbertest
client.

If you use the static version then there is no need to download and
install expat since it's already in jabbertest--thus, my advice is to
just run testsuite-static.

Let me know if this helps.
