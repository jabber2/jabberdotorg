Hopefully everyone has heard by now of the new "Jabber Powered" logo.  This
snazzy new logo is designed to be used far and wide to promote the use of
Jabber.  By using the logo, you'll demonstrate your support for Jabber.  In
exchange, we'll list your site on the Jabber Powered site (and other places)
so Jabber will demonstrate its support for you.  To find out more, go to
http://www.jabberpowered.org.

If you've already got the logo, make sure you let me know you're using it
and I'll make sure your link is up on the web site.
