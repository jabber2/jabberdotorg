Hi all,

I've just completed a Jabber component that allows to you set events
that you would like to be reminded of. At the right time, the server
will send you a message (or anything else) to notify you of this
event.

The module itself seems to work well, but without client support it's
quite tedious. If anyone likes it enough to want to implement it in
their client, that'd be great ;)

If you use this, send comments. I'd like to hear about it! I think this
could be quite a useful module.
