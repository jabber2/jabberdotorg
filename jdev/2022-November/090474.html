Hello,

The XMPP <=> ActivityPub gateway I'm working on is now feature-complete, you can see some 
details in my last progress note at https://www.goffi.org/b/libervia-progress-note-2022-w45-MTdL[1] (available on XMPP and ActivityPub).

However, it's not stable yet, so I would need feedback to report any problems, crash, missing 
message, incompatibility, etc.

I would appreciate if early adopters could test it and report me problems on the Libervia MUC 
(libervia at chat.jabberfr.org[2]). Again it's still considered alpha quality for now, don't use it as 
your main ActivityPub instance, instead it would be good to keep other account to check what is 
possibly msising.

Installation instructions and implementation details are available in the documentation at 
https://salut-a-toi.org/documentation[3], and specifically the installation[4] and components[5] 
parts.

You can either use the integrated CLI frontend, the web frontend (which is not yet integrating 
new features, I'm working on it), or an other blog-capable client (such as Movim), which don't 
integrate every new feature yet either.

feeback welcome

Kind Regards
Goffi
