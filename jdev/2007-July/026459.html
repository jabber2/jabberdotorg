Hi all!

I've released the next version of Net::XMPP2, here
the changes from the perldoc of Net::XMPP2:

   This release adds lots of small improvements to the API (mostly new events),
   and also some bugfixes here and there. The release also comes with some new
   examples, you might want to take a look at the "EXAMPLES" section.

   As a highlight I also present the implementation of XEP-0004 (Data Forms), see
   also Net::XMPP2::Ext for a description.

   I also added some convenience functions to Net::XMPP2::Util, for example simxml
   which simplifies the generation of XMPP-like "XML".

See CPAN for documentation and download:

   http://search.cpan.org/dist/Net-XMPP2/

Greetings,
