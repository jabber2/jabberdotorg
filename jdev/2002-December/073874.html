I am pleased to announce the first release of Jabber::mod_perl.

This is mod_perl (an embedded Perl interpreter) for the new jabberd
session manager (sm) component.

Functionality included is:

* ability to register multiple handler stacks within each chain
  of the session manager
* auto-reloading of modified Perl module handlers
* Perl wrapper class for session manager pkt objects
* Perl wrapper class for session manager nad objects

This all adds up to being able to write session manager plugins directly
in Perl.

This has been uploaded to CPAN and should be on a mirror near you soon
- you can try looking here: http://search.cpan.org/search?query=Jabber%3A%3Amod_perl&mode=module
or here: http://www.piersharding.com/download/Jabber-mod_perl-0.03.tar.gz

Many thanks to Rob Norris for all his assistance.

All commments welcome.

Cheers.
