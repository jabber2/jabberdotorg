I believe that Symbian will be making WV a standard optional feature (like
email and WAP) of the OS so we should see WV support built into many new
mobile phones. Despite protests from one of our list members, I agree that
this is something we should target for interop.

Even if WV ends up being as dead as that poster claims, I think it will be a
useful task that will teach us a lot about making XMPP interop with other IM
systems. Shall we start a working group or something? I'm pretty short on
bandwidth so if someone else could kick start this I can contribute.
Otherwise, I can try to put something together in Jan when I dig out from my
current todo backlog.
