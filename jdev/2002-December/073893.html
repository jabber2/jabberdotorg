I made some experiments with xdb_ldap from jabberstudio.org CVS. I don't know
how to contact the owner of this project and hope he reads this mailing list.

Two comments:

- The included ldap schema defines JIDs as case sensitive - but they are not.
- The configuration example (jabber-xdb.xml) has an element <rootdn>, but
  the xdb_ldap module expects this node to be named <suffix>
