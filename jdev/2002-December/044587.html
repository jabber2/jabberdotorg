It runs fine as an app on my system. It is not designed to be run as a
service on win32 and should not be, the win32 port is in an experimental
state and a few problems have been found with it that are yet to be
addressed in the currently available binary version, as it has already been
said it should not be used on production servers only to aid in development
and testing.
