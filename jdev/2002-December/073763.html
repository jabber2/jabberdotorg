On 12/11/02 6:18, "Ian Sherwood" <sherwoi at yahoo.com> wrote:

> Does anyone know of a Jabber server written in Java? Ideally, I am looking for
> an existing Open Source code stream that I could start from. The goal is to
> integrate a Jabber server with Lotus' Domino server as the persistance
> platform.

If you're willing to do a significant amount of coding there is the JJ
project at jabberstudio.org. It's based on the Apache licensed source in my
book (http://www.manning.com/shigeoka). Of course, I'd recommend buying my
book as a good companion to that source. :) JJ is not finished (it won't
support a standard Jabber client session yet) but should be functional with
1-2 man months of effort depending on the skill of your developers and their
familiarity with Jabber.
