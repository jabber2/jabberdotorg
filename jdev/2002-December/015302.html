JabFoaf is a web-based utility to convert Jabber roster items into FOAF
files. FOAF is the 'Friend of a Friend' project, which is part of the
Semantic Web effort. JabFoaf makes use of the excellent Jabber PHP class
written by Carlo Zottmann.

  JabFoaf: http://dougal.gunters.org/jabfoaf/

  The FOAF Project: http://www.rdfweb.org/foaf/

I hope to have the code cleaned up for public release within a couple of
weeks.
