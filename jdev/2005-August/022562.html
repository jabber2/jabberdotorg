I'm pleased to announce the release of gloox 0.4.

The backend has changed quite a lot and an OO interface as well as server 
certificate verification have been added. Thanks to valgrind, the number of 
memory leaks within gloox SHOULD be down to zarro. Some bugs in the protocol 
implementations have been fixed.
Any feedback is highly appreciated.
