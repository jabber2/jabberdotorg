Loudmouth 1.0 released
Aug 14, 2005

What is Loudmouth
Loudmouth is a small C library for writing Jabber clients written using
GLib. It's designed to be extensible while maintaining ease of use.

News in 1.0
This version fixes a number of issues noticed with the 0.90 beta
release. The most severe being that the SSL fix in that release didn't
work all the times, this has been fixed.

A couple of issues when using the blocking APIs has also been fixed.

Project updates
Would also like to mention that Loudmouth now has a new developer site
at http://developer.imendio.com/wiki/Loudmouth.

Full list of changes:

* [LM-3] - Segfault int connection_in_event
* [LM-6] - m_connection_send_with_reply_and_block() should check if
           the connection is open
* [LM-7] - lm_connection_authenticate_and_block() doesn't set the
           connections state to a proper value
* [LM-8] - lm_connection_open_and_block() blocks infinitely if the
           server counldn't be reached
