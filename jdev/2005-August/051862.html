Katz, Dov B (IT) wrote:

> Does JabberBeans support sasl/google talk's login/security mechanism? If
> not, what woudl need to be done to add such support?

No.

JabberBeans is ancient. For more modern protocol compliance (including 
TLS and SASL, which will enable you to authenticate with Google Talk), 
try JSO or Smack:

http://jso.jabberstudio.org/
http://www.jivesoftware.org/smack/

That said, I understand that you can connect to Google Talk on port 5223 
with plaintext auth using old-style JEP-0078 connections, so JabberBeans 
might work for you (until and unless the Google folks turn that off).
