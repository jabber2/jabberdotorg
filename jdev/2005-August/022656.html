Version 0.5 of the gloox library has been released.
Major changes include full support for xml:lang of Stream as well as Presence, 
Message and Error stanzas, ability to override the JID's host part for a 
connection, XMPP version check, a log handler, re-introduction of 
non-blocking connect.
With this release only very few minor parts of the specs remain unsupported.

http://camaya.net/gloox

gloox is an XMPP library for C++ which aims at being completely compliant (as 
far as it is reasonable for a library) to the XMPP RFCs (both Core and IM).
