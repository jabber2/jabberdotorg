Loudmouth 1.0.5 released!

What is Loudmouth
Loudmouth is a small C library for writing Jabber clients written using
GLib. It's designed to be extensible while maintaining ease of use.

Changes in 1.0.5

    * Infinite reading of 0 bytes when disconnected using gnutls, LM-45.
    * Loudmouth exports some internal SHA* functions, LM-47.
    * Fix auto* warning, LM-46.
