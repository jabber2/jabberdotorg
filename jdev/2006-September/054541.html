Let's say a user's client wants to know whether a contact can receive an
offline message. In http://www.jabber.org/jeps/jep-0160.html section 4, it
talks about how this can be handled through service discovery. In the
example, the discovery is to the server "capulet.com". Let's say a server
has offline mesages enabled for some users but not others. Would it be
correct for it to return the <feature var='msgoffline'/> for to="
juliet at capulet.org" but not for to="nurse at capulet.org"? Would/should clients
even do service discovery to the user's bare jid to find out this info?
