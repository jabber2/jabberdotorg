Greg,

A better forum for asking questions like this one is JADMIN, which
for server installation/configuration questions. See 
http://mailman.jabber.org/listinfo/ for subscription info.

The Jabber Server can run on an isolated or firewalled network. The
errors you are seeing does not imply that your Jabber Server is failing,
only that some additional features aren't being used. See the Jabber FAQ
for more info:
http://docs.jabber.org/general/html/faq.html#AEN233
