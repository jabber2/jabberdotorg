There are many jabber.org mailing lists for other topics. A good 
fraction of the traffic here seems to be from people trying to build 
and/or run the Jabber server -- that's what the jadmin list is for. jdev 
is supposed to be focused more on development, i.e. people trying to 
write code and/or work with the protocol.

You can view and subscribe to all the Jabber mailing lists at:
http://mailman.jabber.org/listinfo/
