Announcing the Jabber Instant Messenger discussion list.

Jabber.com, Inc. has introduced a new e-mail discussion 
list for Jabber Instant Messenger.  The list is intended 
for users of the Jabber Instant Messenger client. Users 
can discuss problems they're having with the client as 
well as features and enhancements they'd like to see in 
future releases of Jabber Instant Messenger.

If you'd like to join the list, please go to
http://support.jabber.com/mailman/listinfo.cgi/jim.
