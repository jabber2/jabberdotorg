I don't know about the others, but this is what I know about ICQ:
ICQ uses a very insecure method for subscription requests, it is
actually handled *client* side! This means that with a quick patch you
can add anyone to your contact list even in the official client. It
would seem a bit pointless for the ICQ gateway to implement this
client side features :)
