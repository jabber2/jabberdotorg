We just released a complete overhaul of the Buddycloud Demo Android app.

It will work against any domain running Buddycloud components
(Buddycloud-server, media-server, push-notification component,
XMPP->REST-api).

The code is at https://github.com/buddycloud/buddycloud-android (gradle
build) and the direct download is at
https://play.google.com/store/apps/details?id=com.buddycloud

Our next steps are to:

   - help new users signup by showing a list of Buddycloud-enabled domains
   - refactor the push notifications receiving side
   - more polishing

Bug reports and pull requests welcomed!

Huge thanks to Adnan for whipping this release into shape.
