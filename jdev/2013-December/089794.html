Hi,

I just wanted to let you know that after 2 1/2 years StropheJS finally
got a new release.
The biggest additions in 1.1.0 are support for XMPP-over-WebSocket and
SCRAM-SHA1 authentication.
Also, the API is compatible with 1.0.2, so upgrading a StropheJS
application should not require any changes besides replacing
strophe.js.
There are a few additions of optional settings though. (see the
documentation: http://strophe.im/strophejs/doc/1.1.0 )

For a full changelog see:
https://github.com/strophe/strophejs/blob/master/CHANGELOG.txt

Regards,

Andreas Guth
