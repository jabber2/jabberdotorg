Hello,

We have just release a preview version of Epeios. Epeios is a module  
that allows to run ejabberd module as components for other XMPP server.

More details are available here:
http://www.process-one.net/en/blogs/article/ 
epeios_write_a_module_for_an_xmpp_server_once_run_it_everywhere/

Yes, it means that a developer can write a clusterable component that  
will run on any XMPP server :)

The code is now delivered with ejabberd pubsub.

Please note that this is a work in progress and as described here:
http://www.process-one.net/en/blogs/article/ 
from_jabber_to_xmpp_im_standard_convergence/
we are working on component protocol improvements to make it even  
more powerful.

Comments are welcome !
