Mediawiki software allows you to define your own inter-wiki links with 
wildcards and leading to any domain. However, if you wanted it to 
indicate (as even Wikipedia does not do for inter-wiki links) by color

not (useful if you wanted an automated link on each page), that would 
have to be programmed.  
http://meta.wikimedia.org/wiki/Help:Interwiki_linking has some
