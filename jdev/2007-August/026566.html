XML 1.1 did not exist when we wrote RFC 3920. The references in both RFC
3920 and rfc3920bis are to XML 1.0. We probably need to add some text to
rfc3920bis about XML versioning. I have not looked at XML 1.1 in quite
some time, so I need to investigate that before writing some proposed text.
