On Wed, Aug 07, 2002 at 10:26:13PM +0200, admin at jabber.fsinf.de wrote:
> Does anyone know some software that implements a kind of Jabber mailing
> lists?

I know of no actual implementation of this concept, and if you desire
to write it yourself, you'd probably want to look at publish/subscribe.

Mailinglists would be an implementation of publish/subscribe as I see it. There
are several proposal protocols for this, but none of them have been approved,
yet.  The proposals can be found at http://www.jabber.org/jeps/jeplist.html as
JEPs 0021, 0024, 0036 and 0040.

Unless you are in a hurry for this, I'd wait until a publish/subscribe is
agreed upon. Probably saves you a lot of headaches.
