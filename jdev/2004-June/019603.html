>From <http://www.jabberstudio.org/news/view.php?id=458>:

   Online project requests are no longer going to be handled by just 
   the form. First, an admin will have to be contacted on Jabber and 
   they will then direct you through the process.

   All pending project requests are in a holding pattern until the
   requestors talk to one of the admins listed on the request page
   (<http://www.jabberstudio.org/project/request.php>).

If you have a project request outstanding on JabberStudio, please
contact one of the listed admins.
