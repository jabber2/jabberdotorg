On Wed, Jun 23, 2004 at 01:00:36PM +0100, Phil Wilson wrote:
> Hi all, 
> 
> Does anyone know of any JEP-0060 (publish/subscribe) implementations other
> that Mimir and PubSub.com?

If you are referring to implementations of a pubsub service, ejabberd seems
to have pubsub support built in. But I don't know the status of this project.
Also, the one at http://pubsub.jabberstudio.org is an old, non-compliant perl
implementation of a pubsub service.

By the way, Mimir is not a pubsub service, it /uses/ pubsub, and currently it
uses Idavoll for the pubsub service. Idavoll is a generic pubsub service. It
is currently being rewritten using Twisted Python, hopefully being more
complete than the current jabberpy version.

Pubsub.com as far as I can see, is tailored to the pubsub.com application,
and not a generic pubsub service anyone can use for their own applications
(Bob, correct me if I'm wrong).

May I ask why you asked?
