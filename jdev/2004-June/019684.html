BTW: Why are you using 'ZONE' as the first argument of log_notice()
calls? ZONE is only intended to be used for log_debug. For log_notice,
log_warn and log_alert the first argument is defined as the host which
sends the log message. (The same problem exists in aim-t and msn-t BTW.)
