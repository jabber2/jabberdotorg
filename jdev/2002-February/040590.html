hello anton.

The jabber.org and jabber.com servers are provided as a free service to 
the jabber commmunity. Jabber.com is based on the commercial jabber 
server. While jabber.org is on the open source server there are quite a 
few differences between the two.

The best way for you to test would indeed be for you to install the 
jabber server followed by the yahoo transport and msn tranport. this 
would allow you to watch how the server responds.

Also there is an excellent jabber programmers guide.available on the 
jabber.com site.


hope this helps.
