Did you make a 'computer.domain.com' directory under jabber './spool'???

i.e. if your jabber computer is 'jabber.company.com' then you must have a directory jabber/spool/jabber.company.com created with access rights compatible with the user running jabberd. That direcotry is where the users are created.
