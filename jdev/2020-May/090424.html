Hello all,

We are pleased to announce the release of slixmpp 1.5.2, which follows the
(unannounced here) 1.5.0 and 1.5.1 versions!
https://lab.louiz.org/poezio/slixmpp/-/tags/slix-1.5.2

1.5.0 was a feature release, which integrates necessary changes for an
external OMEMO plugin, a new XEP-0359 plugin, as well as behaviour
changes such as adding ids to presences by default.

It also introduced many bugfixes and improvements, both in XEP plugins
and internal library behaviour.

1.5.1 Fixed a regression introduced in 1.5.0.

1.5.2 Fixes minor bugs in HTTP Upload and MUC plugins.


-- 
The slixmpp authors
