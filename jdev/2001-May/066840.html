Well the problem you're having is jpolld can't find jabberd.  Have you tried
telneting to jabberd using the same information you are supplying to jpolld?

so from the same machine you're running jpolld 

try telnet 192.0.0.5 5225

If that works then it's a name or password issue, if that doesn't work then
you know that jabberd isn't listening to that port/ip

Hope that helps
Albert
