As mentioned in the server howto on docs.jabber.org, you're getting the
wrong code -- you need to get the 'conference' module, not the groupchat
module (which is old). You can grab it here:

http://download.jabber.org/dists/1.4/final/

BTW, the JDEV list is for Jabber software development -- questions like
the one you're asking are more appropriate for the JADMIN list, you can
sign up here:

http://mailman.jabber.org/listinfo/jadmin/
