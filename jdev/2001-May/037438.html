New Conferencing Protocol Draft Available

The following was posted by akuma at jabber.org via the Jabber DevZone web site (http://dev.jabber.org/):

The Jabber Conferencing Framework is a base for
multi-user systems on jabber; the end-goal is to
have a framework which is extensible enough to
allow for conferencing components to provide any
additional features desired, while still providing
a base for simpler clients to use.



More can be found on the Jabber.org Discussion
Forums 
here, please post any comments on the draft 
in that forum



I am also still looking for someone interested
in aiding me in implementing the new protocol; if
anyone is interested drop me an email at
