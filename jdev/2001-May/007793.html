Hi

we are pleased to announce today the initial release of 
Jabber 1.4.1 for Apple Mac OS X.
As as initial release this port is at an alpha stage, and as 
you will discover in the 'how to' a lot of work still need to be 
done. 
At least different major problems were solved and the 
server can run and accept incoming connections.

You can get more information by visiting :

http://www.funcall.com/Download/Jabber.html

or read the documentation on how to build Jabber for Mac 
OS X at :

http://www.funcall.com/Documentation/Jabber/
BuildingJabberOnMacOSX.html

In order to have the server up and running some 
modifications were done mainly in mio.c, and we will 
appreciate input/feeback on how to improve jabber stability 
on Apple Mac OS X.

Func at ll Jabber team!
