Not exactly an automake, but a little script I like to use to start and stop
jabber. I have jabberd in a bin directory, all the dynamic libs in a lib
directory and all config files in etc. Besides this I have a run directory
and of course the spool directory. The script lives in the root of this
hierarchy.

The script is called in the standard way
jabber {start|restart|stop|status} {icq|msn|yahoo|aim|jud|jabber|*}

This allows me to easily start up the server, restart pieces etc. Just
calling it with start stop etc will start stop all subprocesses.

Hope it is useful for somebody.
Rob
