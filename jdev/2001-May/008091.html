I've updated my watcher system to track Slashdot headlines as well a
freshmeat, if you send a message contain "watch slashdot for linux" to
watcher at jabber.com you'll then get a jabber message back for each article on
slashdot linux as it's topic. It monitors all subject, so you can replace
linux with palm, tech, hardware, etc..... (the full list is at
http://www.slashdot.org/topics.shtml).

I'm interested to hear if;

a) If you think this is useful.
and/or
b) If you think the articles can be presented better.

As a side note, it also stores your resource field when you ask to watch a
topic, so if you ask to watch something from work you won't get the messages
at home. This is another thing I'm open to suggestions about the usefulness
of.
