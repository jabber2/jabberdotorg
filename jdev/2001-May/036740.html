We've ripped out automake for a lot of reasons (although I still
disagree with some).  We wanted a system that can handle any module you
push at it, add it into your configuration, and do a lot of other cool
things like that.  Automake would have required a lot of m4 hacking to
get that going, and I'm not a big m4 fan.  So, we created jsx, but it
was a slow project (a lot since Ryan was laid off).  To sum up, send it
to me and we can possibly start up a dialog for the best way to hook
this all together (I'm the build master).
