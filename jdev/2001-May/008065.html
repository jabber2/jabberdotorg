I've moved my system to use the JID watcher at jabber.com to make it more
memorable (and to avoid problems with ActiveBuddy), and updated it so it can
now handle many sources of info. As a result the command structure has
changed a little, the three commands are now:

1. watch [system] for [information] (example "watch freshmeat for Emacs
JDE").
2. ignore [system] publishing [information] (example "ignore freshmeat
publishing Emacs JDE")
3. list available systems

I'm open to comments on the system, although it must stay as an app that
appears as a client for now (I am following the thread about scalability,
etc.).

I'd appreciate any comments and/or info about possible sources of
information which offer a simple, easy to parse format.

Al.
