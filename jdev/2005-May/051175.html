Loudmouth 0.90 released
May 08, 2005

What is Loudmouth
Loudmouth is a small C library for writing Jabber clients written using
GLib. It's designed to be extensible while maintaining ease of use.

News in 0.90
In this release a lot of fixes from Daniel Lavalliere has been
incorporated. This includes fixing a couple of memory leaks that you
notice when opening and closing several connections. Daniel also
reported and fixed a couple of thread related issues which are
incorporated in this version.

This version should also work with latest GnuTLS by a patch sent from
foser of Gentoo fame.

Project updates
Would also like to mention that Loudmouth now has a new developer site
at http://developer.imendio.com/wiki/Loudmouth.

This release should also be seen as the first pre release of Loudmouth
1.0 which I intend to release shortly. If there is need I will do other
0.9X releases befor getting to 1.0 though.

Full list of changes:

    * [LM-1] - Support GnuTLS 1.2
    * [LM-2] - Fix or remove the Mono bindings
    * [LM-3] - Segfault int connection_in_event
    * [LM-4] - Missing call to release ssl
    * [LM-5] - GLib warnings due to duplicate call
