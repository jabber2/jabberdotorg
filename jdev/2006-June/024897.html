Loudmouth 1.0.4 released

What is Loudmouth
Loudmouth is a small C library for writing Jabber clients written using
GLib. It's designed to be extensible while maintaining ease of use.

Changes in 1.0.4
* Fixed issue where no disconnect signal was emitted when connection
  closed during opening, LM-43
* Plugged a memory leak in LmConnection, LM-39
* Fixed warnings due to uninitialized value, LM-35
* Fixed a reentrancy-related issue with LmConnection, LM-37

news, please join our announce list:
