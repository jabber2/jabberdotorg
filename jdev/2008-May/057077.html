A few weeks ago the XMPP Council had a discussion on removing Section 5
("Publishing Items") from XEP-0030:

http://www.xmpp.org/extensions/xep-0030.html#publish

http://logs.jabber.org/council@conference.jabber.org/2008-04-30.html#14:03:12

This message seeks feedback on the topic.

The disco-publish feature was added in Version 0.12 of XEP-0030, on
2003-02-06. However, as far as I know it has not been implemented
widely. Furthermore, it is not a core use case for service discovery,
and nowadays could be implemented via pubsub/PEP if desired.

Are there any objections to removing this feature from XEP-0030?
