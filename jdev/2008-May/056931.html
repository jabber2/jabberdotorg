There is a new STUN/TURN server implementation:

	http://numb.viagenie.ca

Access is free. You just need to create an account via the web form.
Long-term
authentication is used.

We have implemented TURNv6 but our IPv6 block doesn't have global
reachability
and so IPv6 has been turned off for now. If there are people interested in
that, please do tell us so that we know on which feature to work first.

This is the first public announcement so expect rough corners.
