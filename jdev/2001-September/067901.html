Hallo Edwin! :)

That's cool about your applet. The JOSL is not viral in the way that the
GPL is -- it's quite similar to the Mozilla Public License except that it
plugs a few loopholes in the MPL. So if you make changes to the applet
code itself (whatever you downloaded initially), you need to contribute
those back to the JabberApplet project (which as noted is a project
without a leader -- do you want to take over? :)

Peter
