Hi !

I have a very small patch to to this (attached to this message).
Note that you can use it only if you use the default client listener
(pthsock) and not jpolld or dpsm.

For this patch to work you should add the following to your jabber.xml
config file:

	<log id='mlogger'>
                <host/>
                <logtype>msg</logtype>
                <format>%d %s</format>
                <file>messages.log</file>
        </log>

You'll have a file named 'messages.log' containing all the messages ...
Hope it helps :)

Stefan
