JEP Information Update

The following was posted by julian at jabber.org via the Jabber DevZone web site (http://dev.jabber.org/):



A reminder that a JEP is a Jabber Enhancement Proposal, the new way to
get enhancements to Jabber approved by the Jabber Council, thus
officially part of Jabber.



Peter Saint-Andre posted to the foundation mailing list:



FYI, I've updated the JEPs info on the Foundation website. In
particular,
I've created a template based on which JEP authors can create new
JEPs,
and I've put the existing JEPs into a table on a separate web page so
we
can more easily track their status.



BTW, I have received three new JEPs in the last week or so (avatars,
profile permissions, and Jabber-RPC) and will be placing them before
the
Council in the next few days.
