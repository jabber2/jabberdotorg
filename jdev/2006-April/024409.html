Loudmouth 1.0.2 released
April 5, 2006

What is Loudmouth
Loudmouth is a small C library for writing Jabber clients written using
GLib. It's designed to be extensible while maintaining ease of use.

News in 1.0.2
This version is a bug fix release containing some critical fixes and all
users of Loudmouth is recommended to upgrade.

Full list of changes:

* [LM-15] - lm_connection_open_and_block didn't set error message
            properly.

* [LM-16] - Fixed segfault after error during connect phase.
* [LM-17] - Large messages (>16k) got truncated when using SSL.
