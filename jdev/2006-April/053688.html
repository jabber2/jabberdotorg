Stephen Hsu wrote:
> Hi,
>  
> I wonder what would be needed if I want to develop an IM software based
> on Jabber open source codes?  Do all the source codes support double
> character display?  Very appreciate if someone can help me with the
> answers.   Thanks a lot!!

"Jabber" is a protocol and natively supports only UTF-8. There are many
implementations, not all of which may support UTF-8 (but really they
should and I don't know of any exceptions). If you have questions about
a particular implementation, ask the developers of their codebase.
