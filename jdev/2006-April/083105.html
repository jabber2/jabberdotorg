Yesterday we discovered that a back door had been installed on apollo
(the jabber.org machine that runs our web server). The damage appears to
be limited and we are working to clean the box. The entry point appears
to have been our old friend PHP. A number of PHP-related applications
have been disabled on apollo. This includes the PHP wiki software that
was being used by a number of JabberStudio projects (without, it seems,
appropriate access controls). As a precautionary measure, we have
temporarily disabled all JabberStudio project websites. However, the
main JabberStudio website is still up and running. Also, CVS/SVN access
and the mailing lists remain fully functional, since machines other than
apollo are used to run source control and mail.

I apologize for any inconvenience this may cause, and thank you for your
patience as we work through these issues.

Peter
