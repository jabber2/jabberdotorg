Hi everyone,

I just released biboumi 5.0.

A one-sentence-recap of this release could be:

“Persistent channels and MAM support make biboumi act as an IRC bouncer 
and finally leads to a mobile-friendly experience for IRC”.


You can download it, find the changelog and the documentation on the 
website: https://biboumi.louiz.org/

Here is a blog post I wrote to summarize the most important changes in 
this release: https://blog.louiz.org/biboumi-5.0


For the docker users among you, I also made a biboumi docker image, see 
https://hub.docker.com/r/louiz/biboumi/ if you want to try it.

This should make biboumi’s installation very smooth and easy (as long as 
you already installed and configured an XMPP server, whether in docker 
or directly on the host server).


I hope you’ll like it :)
