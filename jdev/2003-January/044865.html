Just for the record, we are planning on releasing a Gtk2/GNOME 2 client
soon (I just have to find the time to do the final touches to both the
client and the client library).

http://micke.hallendal.net/screenshots/Gossip-groupchat.png
http://micke.hallendal.net/screenshots/Gossip.png

The code is available in our CVS at:

cvs -d:pserver:anonymous at cvs.codefactory.se:/cvs co loudmouth
cvs -d:pserver:anonymous at cvs.codefactory.se:/cvs co gossip

You also need libgtcpsocket from GNOME CVS:
cvs -d:pserver:anonymous at anoncvs.gnome.org:/cvs/gnome co libgtcpsocket

Together with GNOME 2.0 or higher.

Please try it out if you want and file bugs against
http://bugzilla.codefactory.se/ if there are features that you really
need to start using it. Currently support for transports are missing
(though if you already have the transports setup they work).
