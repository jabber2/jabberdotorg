Hi there,

  Ok, heres a suggestion in general if you have a problem and need
  help:

  * Make sure you include versions - what version of jabber and msn-t
    are you using?

  In this case, it may be related to recent changes at Microsoft - if
  you have <server> setting in your config for msn-t, change them to
  only have one, and set it to 'messenger.hotmail.com'

  Alternatively, grab the latest CVS version of MSN-t, and you don't
  need the <server> settings as 'messenger.hotmail.com' is already coded
  into the source code.

Regards,

  David
