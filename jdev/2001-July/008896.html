Libyahoo looks for GTK, because it is a part of the gtkyahoo project.  you can compile libyahoo without looking for GTK by using the --without-gtk configure flag.

Yahoo! transport is using the libyahoo project without any modifications currently, however due to build problems on Solaris, this might change in the near future..
