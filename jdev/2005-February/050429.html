In order to install two new server machines, we are planning some
downtime this weekend for various JSF and JabberStudio services. 
The work will occur this Saturday, February 12, starting around 
16:00 UTC. We do not know exactly how long the migration will 
take, but we will try to limit the downtime as much as possible. 
The services affected will be:

- jabber.org website
- jabber.org IM server (including groupchat etc.)
- jabberstudio.org website (nothing to see right now anyway)
- jabberstudio.org CVS access

The mailing lists will not be affected by this service outage.

My apologies for any inconvenience this may cause.
