Hello,

I just wanted to let you know that we have release a new version of the
XMPP based integration platform which is based on ejabberd: J-EAI 1.0
beta 4.

This new version add many feature improvements and bugfixes. This
version most notably improves the pubsub supports in ejabberd, adding
many node configuration options. It is possible to define persistant
node or not, and the number of items to persist. It is also possible to
define various notification parameters, such as the delivery of the
payload with the notifications or not, notification of subscribers when
some changes occurs on items, maximum items size of the payload
definition, subscription approvers, notification of online users only or
not, etc.

A ready-to-run version has been packaged on the Erlang REPOS 1.1 CDROM
and works out of the box. The CDROM can be downloaded from:

http://downloads.erlang-projects.org/repos/repos-1.1.iso.tar.gz

The code source and some screenshots are available from:
http://www.process-one.net/

The publish and subscribe changes still may have some bugs but we are
doing heavy testing to make the final release very stable.

The J-EAI improvements can directly benefit to ejabberd provides
extended publish and subscribe support for the ejabberd server.
