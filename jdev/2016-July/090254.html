Hello XMPP community!

we have just released Libervia/Salut à Toi 0.6.1, you can see the release note 
at http://www.goffi.org/blog/goffi/07786b84-827d-4f54-8d6f-3e73f053d9d5 (this blog 
is run by SàT).

Salut à Toi is a XMPP based communication tool/social network, which is 
multipurpose and multi frontend (cli, console, web, and a new desktop/mobile 
devices frontend is on its way).
You can have an overview of features at https://salut-a-toi.org/features.html

If you want to try it easily, we have also improved our Docker images, you can 
check the instructions at https://wiki.goffi.org/wiki/Docker/en

Our next release (0.7) is expected to be stable and tested enough to be used 
seriously in production.

feedbacks welcome

Thanks
Goffi
