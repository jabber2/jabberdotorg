If you have implemented XMPP over WebSocket, please do 3 things:

1. Read http://tools.ietf.org/html/draft-ietf-xmpp-websocket-01

2. Provide feedback on the xmpp at ietf.org list (you don't even need to be 
subscribed), even "I've reviewed it and it looks sane" is OK

3. Update your implementation to track the latest spec version

Thanks!
