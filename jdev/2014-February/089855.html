Hi all,

Last weekend I implemented a little add-on module to my Jabber client
(jabber.el) that presents the information that the client receives
through XEP-0115 (Entity Capabilities).  It provides a few different
views into the information available:

* list features supported by a certain client
* compare features of different client versions
* list contacts that use a certain client version
* list clients that support a certain feature

I haven't seen anything similar, so I figured this might be interesting
to the list.  Screenshots and download links are available at
http://legoscia.github.io/jabber-caps-browser/ .
