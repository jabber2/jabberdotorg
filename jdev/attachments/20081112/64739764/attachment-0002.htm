<p class=MsoNormal>Hi All (esp. PSA <span style='font-family:Wingdings'>J</span>),<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal>One thing that I have seen with my experiences so far is
that there is practically <i>no</i> formal component protocol. &lt;<a
href="http://xmpp.org/extensions/xep-0114.html">http://xmpp.org/extensions/xep-0114.html</a>&gt;
only shows us how to establish a stream. Things that nag me (which I still have
to figure out, so if you have answers...):<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoListParagraph style='text-indent:-18.0pt;mso-list:l0 level1 lfo1'><![if !supportLists]><span

</span></span></span><![endif]>No indication on newer authentication protocols,
just handshakes (don&#8217;t need answers here, I can take example from other
server docs, FYI Tigase has really good docs in regard to this)...<o:p></o:p></p>

<p class=MsoListParagraph style='text-indent:-18.0pt;mso-list:l0 level1 lfo1'><![if !supportLists]><span

</span></span></span><![endif]>Binding which is mainstream in, from what I can
tell, all current XMPP servers. This is not in the XEP.<o:p></o:p></p>

<p class=MsoListParagraph style='text-indent:-18.0pt;mso-list:l0 level1 lfo1'><![if !supportLists]><span

</span></span></span><![endif]>Routing stanzas, again, are not in the XEP.<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal>Here comes the big one: given that binding is for all
intents and purposes &#8216;official&#8217;, how do I handle these scenarios:<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoListParagraph style='text-indent:-18.0pt;mso-list:l1 level1 lfo2'><![if !supportLists]><span
style='font-family:Symbol'><span style='mso-list:Ignore'>&middot;<span
style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span></span><![endif]>I want to send a message directly to &#8220;router15&#8221;
(one of the components) and not have it leave the network.<o:p></o:p></p>

<p class=MsoListParagraph style='text-indent:-18.0pt;mso-list:l1 level1 lfo2'><![if !supportLists]><span

style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span></span><![endif]>I want to send a message to all components of a
specific type (related to the above).<o:p></o:p></p>

<p class=MsoListParagraph style='text-indent:-18.0pt;mso-list:l1 level1 lfo2'><![if !supportLists]><span

</span></span></span><![endif]>I want to send a message to a user.<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal>My current best guess is (these are embedded in the body of
stream:stream):<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal>&lt;!-- send message to specific component --&gt;<o:p></o:p></p>
