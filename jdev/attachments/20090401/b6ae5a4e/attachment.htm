<p class=MsoNormal>Hi All,<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal>I thought someone might want to do something interesting
with this technology.<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal>&quot;&quot;APML allows users to share their own personal
Attention Profile in much the same way that OPML allows the exchange of reading
lists between News Readers. The idea is to compress all forms of Attention Data
into a portable file format containing a description of ranked user interests.&quot;&quot;<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal>Given an APML binding for each PubSub node (e.g. &#8220;user
interface&#8221;) a PubSub service could provide a user with a list of
suggested nodes based on an APML document that they provide (via IQ or
something): or automatically send them publications in any node based on each
publications&#8217; tags.<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal>Furthermore, if the PubSub spec got a little &#8216;smarter&#8217;
the author could set the &#8216;affinity&#8217; for each post&#8217;s tag:
allowing them to indicate how well the post relates to the particular tag. This
could be used to filter publications based on the &#8220;value&#8221; attribute
in the APML. Thus:<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal>Known APMLs:<o:p></o:p></p>

<p class=MsoNormal>Fred:<o:p></o:p></p>

<p class=MsoNormal>&nbsp;Food 0.6<o:p></o:p></p>

<p class=MsoNormal>&nbsp;User Interface 0.9<o:p></o:p></p>

<p class=MsoNormal>Jane<o:p></o:p></p>

<p class=MsoNormal>&nbsp;Food 0.8<o:p></o:p></p>

<p class=MsoNormal>&nbsp;User Interface 0.3<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal>Subscriber Creates Publication:<o:p></o:p></p>

<p class=MsoNormal>&nbsp;Food (3/5 = 0.6 &#8211; Invert = 0.4)<o:p></o:p></p>

<p class=MsoNormal>&nbsp;Sends to &nbsp;Fred and Jane<o:p></o:p></p>

<p class=MsoNormal>Another one:<o:p></o:p></p>

<p class=MsoNormal>&nbsp;User Interface (2/5 = 0.4 &#8211; Invert = 0.6)<o:p></o:p></p>

<p class=MsoNormal>&nbsp;Sends to&nbsp; Fred<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>
