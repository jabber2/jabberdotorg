What is agsXMPP?<br>
agsXMPP is a SDK / library for the eXtensible Messaging and Presence<br>
Protocol (XMPP) protocol written in managed C# dedicated to .NET and<br>
Mono technologies.<br>
<br>
Its designed for:<br>
* Microsoft .NET Framework (1.1 - 3.5)<br>
* Microsoft .NET Compact Framework (Pocket PC, Smartphone, Windows CE)<br>
* Mono<br>
<br>
The SDK is released as open source under a dual license.<br>
