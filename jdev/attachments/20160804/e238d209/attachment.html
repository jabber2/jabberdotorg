Biboumi is an IRC gateway. Its primary goal was (and still is) to be very reliable and verbose about what’s going on from the IRC side. For example, I wanted a gateway that would announce the status of the connection to the IRC server and be verbose about the error (failed to resolve the hostname, still trying to connect, could not connect because it doesn’t speak TLS on that port?). Also the user should know what’s happening with their messages (for example if you’re banned, or the channel is in +m mode) instead of just having them being dropped without any warning.<br>
<br>
This release adds a feature that was asked by many: multi-session nicks. i.e. you can now join an IRC channel behind one single nick, from multiple XMPP resources at the same time.<br>
<br>
More details about this release can be found on my blog <a href="https://blog.louiz.org/biboumi-3.0" rel="noreferrer" target="_blank">https://blog.louiz.org/biboumi<wbr>-3.0</a><br>
