<p dir="ltr">Because it IS a subscription list, some clients will still list the user if you haven't unsuscribed from them if you show offline users (pidgin for example).</p>
<p dir="ltr">However by unsubscribing from a user you stop seeing their online presence, and some servers by default won't allow messages to be sent to someone you aren't subscribed to.</p>
<p dir="ltr">So, check you are showing offline users is about the best you'll get.</p>
