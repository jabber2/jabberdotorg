<P><FONT SIZE=2>I was able to build the server (1.2) using gcc on AIX 4.3.3, by</FONT>
<BR><FONT SIZE=2>using the '-G' linker flag for the shared objects (to allow</FONT>
<BR><FONT SIZE=2>symbols to be left unresolved until needed by the runtime linker).</FONT>
<BR><FONT SIZE=2>Then the main executable, jabberd, needs &quot;-brtl -bexpall&quot; to use</FONT>
<BR><FONT SIZE=2>runtime linking and export its symbols to these other objects.</FONT>
</P>

<P><FONT SIZE=2>If you are using gcc, you can add this to the configure file</FONT>
<BR><FONT SIZE=2>where it has a case statement for different OS's:</FONT>
</P>

<P><FONT SIZE=2>&nbsp;&nbsp;&nbsp; AIX)</FONT>
<BR><FONT SIZE=2>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; CFLAGS=&quot;$CFLAGS -DMAXDNAME=1025&quot;</FONT>
<BR><FONT SIZE=2>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; MCFLAGS=&quot;$MCFLAGS -Wl,-G&quot;</FONT>
<BR><FONT SIZE=2>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; XLDFLAGS=&quot;$XLDFLAGS -Wl,-brtl -Wl,-bexpall&quot;;;</FONT>
</P>

<P><FONT SIZE=2>Then rerun configure, and remake; it should build to completion.</FONT>
