Years ago, Carlo Zottmann handed off maintenance of Class.Jabber.PHP.&nbsp; I released several bugfixes, but I never really did much with it.&nbsp; PHP5 was coming out, and I new a rewrite was in order to take advantage of more of the OOP features.&nbsp; I promised a rewrite that never came, until today.<br>
<br>XMPPHP is designed to work with XMPP 1.0 servers, event driven, has TLS encryption and other goodies.&nbsp; You can connect to any XMPP 1.0 server, including <a href="http://jabber.org">jabber.org</a>, Google Talk, LJ Talk, etc.&nbsp; Currently, it&#39;s missing Roster methods and In-Band Registration (those will be coming in the next few days).&nbsp; Also coming is BOSH support, object based stanza generation, etc.&nbsp; Please download it and let me know what you think.&nbsp; I can be IM&#39;d at <a href="mailto:fritzy@netflint.net">fritzy@netflint.net</a>.<br>
<br>This library has nothing in common with Class.Jabber.PHP save for a few lessons learned.&nbsp; It is largely based on my Python library (SleekXMPP).&nbsp; I&#39;m taking things that work well in each and applying them to each other.&nbsp; I expect this to be beneficial for both projects.<br>
