<P>Invisibility support is available in the latest server.<BR>&nbsp;<BR>There are two ways to use Jabber invisibility: <BR><STRONG>on a per-buddy and on a per-server basis. </STRONG></P>
<DIV><STRONG></STRONG>&nbsp;</DIV>
<DIV><STRONG>To use per-buddy invisibility: </STRONG><BR>One can be visible to all others buddies and can set his invisibility <BR>Only to a particular user if he wants. This can be done be done by choosing the buddy name to whom the user wants invisible and send a invisible message and the user wont be visible to that particular buddy, and he is online to all other users. This is done by sending "Do not Distrub" Tag to the corresponding user.</DIV>
<DIV>The "Temporarily" part is a dead give-away. Your invisibility to that buddy will last only until the next time you log on or until you set server invisibility (discussed next) on and off. </DIV>
<DIV>&nbsp;</DIV>
<DIV><STRONG>Server-Invisibility:</STRONG><BR>This is similar to&nbsp;Invisible mode, that will be followed in most of the messengers especially . The user is invisible to all the users.One can set yourself invisible per-server and subsequently make yourself visible to selected buddies only. This is also can be done by sending the "do not distrub" presence message to the server immediately after the user authentic is over.</DIV>
