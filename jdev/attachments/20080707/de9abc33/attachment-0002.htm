<p><span style='font-size:11.0pt;color:#1F497D'>&nbsp;</span><o:p></o:p></p>

<p><span style='font-size:11.0pt;color:#1F497D'>BOSH is what Javascript
libraries would use to communicate with a XMPP server (remember, Javascript
cannot send raw data over the network, all comms must occur over HTTP).</span><o:p></o:p></p>

<p><span style='font-size:11.0pt;color:#1F497D'>&nbsp;</span><o:p></o:p></p>

<p><span style='font-size:11.0pt;color:#1F497D'>What this means to you is that:</span><o:p></o:p></p>

<p style='text-indent:-18.0pt'><span style='font-size:11.0pt;color:#1F497D'>1.&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
No client set up is needed. The javascript libraries (should) include
everything you need. They simply use a special kind of AJAX to communicate with
the server.</span><o:p></o:p></p>

<p style='text-indent:-18.0pt'><span style='font-size:11.0pt;color:#1F497D'>2.&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
You need to find a server that supports BOSH connections. Maybe someone here is
running one that does? You can try to connect to <a href="http://jabber.org/"
target="_blank">jabber.org</a> and mabber or such should also have BOSH
support.</span><o:p></o:p></p>

<p><span style='font-size:11.0pt;color:#1F497D'>&nbsp;</span><o:p></o:p></p>
