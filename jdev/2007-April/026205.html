Nahuel ANGELINETTI schrieb:
> And 2 last questions, in Roster table : the from field is when the user
> accepted the roster, or the roster accepted the user ?

from is if the subscription type is "from", to is if the subscription 
type is "to", and both are set if the subscription type is "both".

subscription type "from" means: there is a subscription from this 
contact (= other user sees owning user)
subscription type "to" means: the user is subscribed to this contact's 
presence (= owning user sees other user)

> and what is ask field ?

If a subscription request has been sent but the subscription is not yet 
confirmed or denied. (Both directions.)
