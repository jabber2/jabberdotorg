On Thu, Apr 14, 2005 at 03:37:57PM -0300, Hernan Tylim wrote:

> I had (have?) a project on Jabberstudio, Neos, but seems that it 
> disappeared from the projects page and I can not access it any more from 
> my account. Is this a temporary glitch from JabberStudio or I need to 
> add it again ? 

You need to request that your project be re-activated. However, last
year temas and I changed the policy so that JabberStudio is hosting only
open-source projects that place their code in CVS/SVN there (i.e., no
more offsite projects). So if you just want to announce new releases for
your project, see below (IIRC, Neos is a client).

> (BTW the request project link: 
> http://jabberstudio.org/newproject.php, is broken)

To request a project, ping one of the JabberStudio admins (e.g., me).

> P.S: Yesterday I made a new release of Neos and I just wanted to make 
> the announcement.

A good place to announce client releases is the JUser list:

http://mail.jabber.org/mailman/listinfo/juser

/psa
