Hey all,

Jive Messenger 2.1.3 has been released with a long list of improvements
and bug fixes. Highlights of the release are large performance
improvements, additional translations, and an improved API for those
writing plugins.

The full changelog is at:
http://www.jivesoftware.org/builds/messenger/docs/latest/changelog.html


About Jive Messenger
--------------------

Jive Messenger is an Open Source (GPL) XMPP server implemented in pure
Java. It works with a large number of databases, and has a powerful but
easy to use web-based admin console. More information and downloads for
Windows and Unix/Linux are available at:
http://www.jivesoftware.org/messenger/


Related News
------------

* An initial version of the Registration plugin has been released by
Ryan Graham. It currently supports admin notification of newly created
accounts and welcome messages. A full list of plugins is at:
http://www.jivesoftware.org/messenger/plugins.jsp

* Picopoint has released a RADIUS module for Jive Messenger, which
allows authentication using the RADIUS protocol. Discussion and download
link at: http://www.jivesoftware.org/forums/thread.jspa?threadID=14342



Regards,
-The Jive Messenger Team
