Hello,

a new release of ejabberd 0.9 has been published. ejabberd is a
clustered, scalable, robust and full featured XMPP server.
This release contains several new major features, such as:
- Virtual hosting,
- Shared roster,
- Experimental relational database support

and includes many bugfixes, architectural changes for performances
improvements.


This release is now hosted on Process-one website
(http://www.process-one.net/). The community site is at:
http://ejabberd.jabber.ru/

ejabberd 0.9 can be downloaded from:
    http://www.process-one.net/en/projects/ejabberd/download.html

For more information, you should have a look at the release note:
    http://www.process-one.net/en/projects/ejabberd/release.html

Have fun,

-- 
The ejabberd team
