Hi Tom,

I tried. It works fine for me. 
Because jivesoftware.org isn't on default server list. You  need
select "Other Server" at the bottom of server list,

Enter "jivesoftware.org" in gray area "Other Server" field. put your
node in username field, no "@jivesoftware.org". Such as my JID is
"wjn at jivesoftware.org". only need put "wjn" in username field. I will
change it latter, allow to enter full JID or just node.

If you like, you can add jivesoftware.org in servers.xml under
WEB-INF\classes and change signin.jsp, replace "jabber.org" to
"jivesoftware.org", so defualt server will be "jivesoftware.org".

If you still have problem, let me know. Thank you for using WebJabber.
