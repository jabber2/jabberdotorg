> I'm having a rather unproductive night, trying to build the jabberd2 alpha 
> on a freebsd 4.6.2 system. 
> 
> The main problem I have encountered is the configure script complaining 
> that it can't find libSASL. 
> 
> I have installed both cyrus-sasl and cyrus-sasl2 from the ports tree, but 
> configure still cant find the libraries aftoer doing so. Also had some 
> trouble with SASL complaining about berkely DB at some stage tonight.. 

cyrus-sasl2 is the one you want. You will also need to install
development versions of thes packages (I don't know if ports seperates
them in the same way most Linux distros do).

The library should be called libsasl2, and should be found on your
default system library path.

If you have something different to this, you'll need to modify
configure.in and rebuild.
