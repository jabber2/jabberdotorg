Hello, all, Adam Theo here.

I've set up a new JabberStudio project to store icon styles at 
<http://www.jabberstudio.org/projects/icon-styles/project/view.php>, 
with the home page at 
<http://www.jabberstudio.org/wiki/index.php/Icon_Styles>. Icon Styles 
are from JEP 0038, and Jabber's answer to emoticons. It is a cool 
feature that no other IM system has.

If there are any graphical artists on the list, it would be great if you 
could take a bit of time to create some emoticons or generic icons for 
use in an icon style. Think of a unique theme for your icons, such as a 
golden, angelic-themed set of emoticons, or a metallic, industrial 
style. As long as you can create the free graphics, there's someone who 
can put them into icon style packages if you don't want to bother with 
the spec.

Thanks. Let me know in Jabber or email what you want to do for Jabber's 
new icon styles spec.
