I've submitted a message log component that i've been working on to 
jabberstudio.  The URL is 
"http://www.jabberstudio.org/projects/msglog".  It's basically a 
component that attempts to modularly support various log targets 
(currently u can log the XML to a log file, mirror other peoples' 
messages to yourself (snooping) and log to a database (currently MySQL 
specific).  I built this for experimental purposes, so it's not 
polished (nor is the code completely structured the way i would like it 
to be), so shouldn't be used in production environments.  Hope someone 
can find this useful in one way or another.

I'm still trying to figure out how to use jabberstudio right now, so I 
haven't put up a Release yet, but you can download the perl file and 
the readme from CVS.

Thanks.
