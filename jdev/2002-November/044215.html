> Also, do persistent rooms created with Exodus by the admin still persist
> if <dynamic/> is set in jabber.xml?
> 
Ok, when you have the dynamic tag set, the persistent option is removed
from the room configuration form (unless you are a sadmin and using the
latest cvs version)

This means that normal users can only make temporary rooms, which
disappear after 5 minutes of being empty. Rooms set persistent by a
sadmin, or rooms made via roommaker, will still be persistent.
