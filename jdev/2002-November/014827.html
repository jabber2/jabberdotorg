It's the first alpha release. There probably won't be a final release

> Does that mean that all modules for Jabber 1.x are no longer working
> with Jabberd 2.0 ? I checked the source code and found it might be
> the fact. I wish somebody can tell me that I am wrong.

JSM modules will need to be rewritten. Internal (library load)
components MAY work by running them inside a 1.4.2 instance in uplink
mode (XDB storage and logging is not available). External components
will continue to work.
