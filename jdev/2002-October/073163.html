Hi,

when I did my post about "file sharing" I started to wonder what the
possibilities of the roster are...

Does the roster as it is now provides a flexible, extensible way of hiding
services for certain users? As I don't understand this question myself ;),
I'll try to give some examples:

In the case of file sharing, it can be that I want person B at server.org to
download file X and Y, but not file Z. person A at server.com can download
X, Y and Z. Sorry for person C at server.com, but he can't even see that I'm
running a file share service...

Of course, this doesn't stop at file sharing: the same ideas goes for
webcams and other, not existing yet services. Maybe that I even want to
hide information from the roster itself, maybe I don't want my "not so
close friends" to see my telephone number...
