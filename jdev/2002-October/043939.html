Just a few notes.  Matt "linuxwolf" Miller, Justin "infini" Karneges,
and myself have been delving deeper into file transfer of late (although
it was put on hold for the birth ;-]).  I'm hoping that we can get some
of our differences resolved in what file transfer, stream negotiation,
and streams actually are, within two weeks.  Currently we're trying to
just look at file transfer and leave file sharing and then queryable
file sharing to other jeps.  I'll try and post a more complete update
soon.

details.  File transfer itself is rather basic =)
