First, JIM is a deprecated product (Jabber Inc. has a new client they have
created).

Second, those features worked only with an older version of Jabber Inc.'s
text conferencing component.

Third, we will soon have support for a much wider range of groupchat
features in Jabber, and Jabber Inc. will be using the same protocol as
everyone else for this. Refer to JEP 45 for details:

http://www.jabber.org/jeps/jep-0045.html
