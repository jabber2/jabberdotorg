> 
> i tryed and i dont received offline messages:(( probably mod_pending isn't use some standard jabber instruction

mod_pending does an xdb call to get everything stored in the
"jabber:x:offline" namespace. Offline messages are stored in this
namespace.

The only thing I can think of is that xdb_sql is storing messages in
some other way. This seems odd to me, but I don't have any experience
with it, and can't help. mod_pending was only ever tested against
xdb_file.
