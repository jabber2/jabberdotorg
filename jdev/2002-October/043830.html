Hi Adrian,

The way I see it - but I'm just newbie with Jabber - what you want to do is basically switch between chat and conference, transparently for the users. I don't think it is possible with the current specs.

At most, if you are designing your own client, you can use chats even for one-to-one discussions, by creating ad-hoc rooms for each such discussion. Of course, your client will have to test if conference is installed on the server, if users are allowed to create new rooms etc. And I think this method might also increase the server load.
