Actually, I believe components will be described on another page
altogether (-:

As for this listing, here's my suggestions:

1)  Add a column for the supported architectures/platforms (e.g.
Linux/x86, Win32, J2SE 1.4, .NET, etc).
2)  Add a set of three columns, one for each "target" (client,
component, server).
3)  (if desired) Add a column for the "completeness", using the current
(stable) version number.
