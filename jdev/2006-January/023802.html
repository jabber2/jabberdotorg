Not that I completely agree with this quote, myself.  It's responsible 
for the majority of the damage to the web caused by Internet Explorer, 
for a start.

My own opinion is that if what you send isn't even compliant to the 
standard, it should be dropped flat and an error spewed back.  If web 
browser developers had done this from the start, we wouldn't have a 
single malformed web page in the wild today because the person who 
created the page would have been unable to test it in their own browser.
