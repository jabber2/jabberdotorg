Online Game Group is pleased to announce the release of libstrophe 0.7.2

This is primarily a bugfix release; several error states were cleaned up
and session initiation is now completed.  See 
http://onlinegamegroup.com/projects/libstrophe/file/trunk/libstrophe/CHANGES.txt
for a complete list of changes.

libstrophe is a C-based XMPP library for writing clients and components.
It is lightweight and portable (meaning that it compiles out of the box
on Win32, OS X, and various unices) and is implemented per RFC3920 and
RFC3921.  It is available under a dual GPL and commercial license.

This version is our third public release and is still in active 
development.  The library is usable, but our primary goal with this
release is to start getting wider testing and feedback, especially 
as we're starting to develop some of the more high-level features.


To obtain more information on libstrophe or to download
the latest release, please visit:

	http://onlinegamegroup.com/projects/libstrophe

We hope you enjoy it,
jack.
