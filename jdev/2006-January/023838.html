There is probably nothing wrong on your side, virtual hosting makes things 
more complicated, combine this with the lack of an experienced web developer 
that contributes to ejabberd, and you get such usability issues :-)
So:
1) Click on Virtual Hosts
2) Click on the Virtual Host you want to configure things on.
3) You get these options
The reason for hiding these options is that you can have different admins for 
each virtual host (and also "Über-admins" that can configure everything).

So feel free to provide patches to make the web interface more 
userfriendly ;-)
