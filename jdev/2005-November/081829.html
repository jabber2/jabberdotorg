Hi,

This is to announce the availability of Shutup, Vislab's prototype
AccessGrid-over-Jabber client.  For more information about the
AccessGrid see http://www.accessgrid.org/

The code is available via Subversion, to get it use:

    svn co svn://projects.vislab.usyd.edu.au/shutup

You also need pygtk and twisted.  To help out new users I've produced a
short video tutorial on installation and usage:

    http://people.vislab.usyd.edu.au/~ssmith/video/shutup-intro.mpg

Feedback, contributions and questions are welcome.

For background on the rationale behind this please see the paper and
tutorial from this year's APAC conference:

    http://people.vislab.usyd.edu.au/~ssmith/apac05/

Cheers,
Steve
