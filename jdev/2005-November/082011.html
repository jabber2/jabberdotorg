Update regarding ejabberd! :-) The contributions page on ejabberd's website 
got its 16th contribution today: a patch for stream compression support in 
ejabberd.

If you want to test the patch, or if you know a client which has support for 
JEP-0138, you can add a comment to next page:
http://ejabberd.jabber.ru/zlib

Test results of this experimental patch and eventual improvements are always 
welcome. Of course, other contributions are also welcome.

Remark that ejabberd is now probably the first open-source Jabber server with 
stream compression support. :-)
