Hello all,

Sorry for two announcements in one day, but there's a lot happening at
jivesoftware.org. :) Besides the launch of Spark, we also released:

 * Jive Messenger 2.3.0 (Open Source XMPP server) -- this release
focuses on XMPP protocol compliance. TLS and SASL for c2s is now
supported. There have been many ease-of-use enhancements and we've also
snuck in a few other goodies such as a framework for ad-hoc commands.
The full changelog is at:

http://www.jivesoftware.org/builds/messenger/docs/latest/changelog.html

 * Asterisk-IM 1.0 (telephony integration between Asterisk and XMPP)  --
after a long beta process, the 1.0 version is now available. Asterisk-IM
1.0 includes many presence integration fixes since the last beta. The
full changelog is at:

http://www.jivesoftware.org/messenger/plugins/asterisk-im/changelog.html

 * Smack 2.1.0 (Java client API for XMPP) -- we built Spark on top of
Smack and rolled in many fixes and improvements from that process for
the 2.1 release. Unfortunately, that meant some API changes, so be sure
to read through the changelog before upgrading:

http://www.jivesoftware.org/builds/smack/docs/latest/changelog.html


Finally, those visiting will notice an updated website look and feel.
We've tried to spruce up the look while also making information better
organized.

Regards,
Matt
