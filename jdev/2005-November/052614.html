The jabber.org server runs a mish-mash of software -- jabberd1.4.x for
the core router and the connection manager from Jabber Inc. See
http://status.jabber.org/ for details.

We have some homegrown scripts that we use to monitor incoming/outgoing
ports, figure out the number of registered users, etc., then pull that
together with PHP scripts to create the status webpage. It's not really
a very releasable "product". ;-)
