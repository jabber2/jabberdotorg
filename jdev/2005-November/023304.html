Jive Messenger 2.3.0 beta 2 has been released. This version includes some 
improvements such as simplified setup process, new features such as vCard 
and LDAP integration and many bug fixes. Read the change log to learn the 
complete list of changes.

You can download the server from http://www.jivesoftware.org/downloads.jsp

http://www.jivesoftware.org/builds/messenger/docs/2.3.0.beta2/changelog.html

http://www.jivesoftware.org/builds/messenger/jive_messenger_src_2_3_0_beta_2.zip
