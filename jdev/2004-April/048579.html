Just run the transport instance of jabber with the "-D" flag, e.g:

	exec pgrphack setuidgid yahoot \
       	 /usr/local/jabber/jabberd/jabberd \
       	 -c /usr/local/jabber/yahootrans.xml \
       	 -D 2>&1
