Hi all,

Thanks for Florian Jensen and Uber we'd like to invite everyone attending
the XMPPUK meetup tomorrow (4th March 2014) to a free executive transport
ride to Mozilla London.

When signing up for Uber simply enter the promotion code: XMPPUK

The code gives you access to £20 off the user's first ride.  This should
cover the fare between the Hilton Metropole and Mozilla London in both 'X'
and 'EXEC' classes. If you'd like to travel in luxurious style ('UberLUX')
then it may cost slightly extra but only somewhere on the order of £5. (of
course if travelling from elsewhere please feel free to use too).

You can check the Uber fare estimator on their website:
https://uber.com/london

Upon arrival Surevine will have pizza and beer waiting (around 6:30pm). The
latest schedule is posted up on http://lanyrd.com/2014/xmppuk/.

It also seems like I can make the event so will be heading up to London
tomorrow afternoon, hopefully see many of you there.

Regards, Lloyd.
