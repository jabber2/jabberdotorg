Also, make sure that you are using the latest version of JabberCOM. Early
versions were not UTF encoding the XML correctly. Also, instead of sending the
message encrypted in the <body> tag, you could use an <x> tag to send the
encrypted data. This is probably a better method anyways in case any encrypted
msgs get sent to a client that doesn't interp the encryption scheme correctly.
