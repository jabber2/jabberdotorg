Hi List,

just fyi: I sent today a jabber:iq:version query to all Jabber-Servers
and -Components which appeared in the s2s-Logs of jabber.ccc.de during
the last hours, 542 in total, to figure out the popularity of the
different jabberd implementations and gateway distribution. Here are
some rough rundowns:

 jabberd1 => 168
 ejabberd => 109
 jabberd2 => 78
 unknown => 64
 Wildfire => 49
 jsm => 20 (I guess this is pre-1.4.3 jabberd1) 
 MSN Transport => 13
 ICQ Transport => 12
 AIM Transport => 5
 Yahoo! Transport => 5
 JIT => 4
 MU-Conference => 2
 Jive Messenger => 2
 XMPPPY Yahoo! Transport (Dev) => 2
 dotJ => 1
 XCP => 1
 SMS - Transport => 1
 ejabberd mod_proxy65 (unofficial) => 1
 RSS component => 1
 Jobble transport => 1
 psyced => 1
 Gadu-Gadu Transport => 1

Most popular jabberd Version is jabberd 1.4.3 with 115 replies,
followed by ejabberd 1.1.1 with 47 and ejabberd 1.1.0 with 40 replies.

Most popular server OS is Linux 2.6 with 171 setups, followed by
Linux/unknown and Linux 2.4. 

Most "outdated" versions were ejabberd 0.7.5 and jsm 1.1.3.

Using just the public jabber-servers reveals pretty much the same, the
single interesting difference is that Wildfire is only used twice
within 117 servers.
