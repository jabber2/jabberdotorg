Cedric Hyppolite wrote:
> Hi,
> 
> I am wondering if it's possible to discover the Jabber server federation
> starting by only one jabber server.

Right now we have this:

http://www.jabber.org/servers.xml

That data is based on an old feed. We need to update it so it uses the
data from http://www.xmpp.net/ (at which time we'll probably move the
feed from jabber.org to xmpp.net).

But we need to clean up the xmpp.net data a bit first. That should
happen soon.

This makes it possible to discover some subset of the servers on the
public network (not all servers are registered with the XMPP Federation
at xmpp.net). And you can do so via HTTP rather than XMPP (i.e., before
you ever connect to the XMPP network). We could also make this data
available via XMPP if we wanted to (i.e., send a disco#items request to
a certain node at xmpp.net, or just to that server itself), but we have
not enabled that yet.

Hope this helps.

Peter
