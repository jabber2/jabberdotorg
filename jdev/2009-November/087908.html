Wow, that was quick. Unfortunately some users identified a critical
bug in the roster storage in our 0.6.0 release. We've identified and
fixed the problem as soon as we can, and strongly recommend that
everyone running *0.6.0* back up their data directory[1] (as always!)
and upgrade to 0.6.1 at the earliest opportunity.

Prosody is a lightweight Jabber/XMPP server written in Lua. It aims
to be flexible, easy to extend, and simple to use for both users
and developers alike.

The following is a summary of changes since the previous version:

  * Fix critical issue causing non-deterministic roster saving

Apologies to those affected by the issue, if you have any questions
or trouble upgrading then don't hesitate to contact us by any means
listed at http://prosody.im/discuss

[1] http://prosody.im/doc/data_storage

Download: http://prosody.im/download/

Happy Jabbering,
  The Prosody Team
