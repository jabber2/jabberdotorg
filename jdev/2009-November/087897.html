Position Title:  XMPP Protocol and Operations Engineer

 

Company:  SocialVision, Inc.

SocialVision is an angel-funded, innovative Berkeley-based startup whose premier product helps make online video a social experience.  We help media companies monetize their existing content and increase ROI for advertisers.  Our mission is to make online television more fun and interactive by creating new social experiences around it.  We’re rapidly growing and working aggressively to meet pivotal timelines for major customers. 

Responsibilities:

*	Architect, setup and maintain XMPP deployment locally and potentially in the cloud.
*	Create reliable, high availability XMPP server environment that supports over 1 million concurrent users logged in and chatting over MUC. Create monitoring and alerting system for deployment.  Create plan for further scaling as requirements increase.
*	Load test server setup to match performance requirements
*	Help architect and build the chat-centric data collection and analytics, write module or extension to XMPP server to execute this.

·         Extend chat server and MUC component code for our custom functionality and protocol extensions

 

Qualifications:

*	Deep understanding of XMPP protocol and XEPs such as BOSH, MUC, Gateway Interaction, External Components, etc.
*	Ability to scale a (custom) XMPP deployment to over 1 million concurrent users!
*	Good knowledge of cloud computing platform such as EC2
*	Ability to perform load testing of XMPP server according to our custom message patterns
*	Experience hacking on / extending / configuring open source XMPP servers (e.g. Tigase, ejabberd)

Benefits

·         Possibility of ownership/equity of a rapidly growing company.  

·         Casual work environment with great flexibility.​ 

·         Exposure to a wide range of technologies offering tremendous professional growth.​
