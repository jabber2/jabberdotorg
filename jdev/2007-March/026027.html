Dnia 05-03-2007, pon o godzinie 17:43 +0000, Mark Morgan napisał(a):
> Is there any way for an external jabberd 1.4 component to bind to a
> resource?

External components binds whole domains and receive all packets directed
to it.
If you want to handle packets directed to a specific resource, you need
to check it component level. Just check if the "to" resource in packet
is a resource you're interested in. :-)
