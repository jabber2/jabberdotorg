This is quite non-standard use case and I work on implementation
for that in Tigase Jabber server.
Basically, in Tigase it is possible to bind resource from external component
and also "tell" the server what kinds of stanza should be forwarded to
this resource.
This is used to route VoIP stanzas to external component.

If you are interested look at http://www.tigase.org/ site.
This is however Java based Jabber server....

Let me know if you are interested in more details.
