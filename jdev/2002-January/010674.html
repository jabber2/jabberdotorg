Oh, and to make it work on hp-ux, too...

Following the instructions (well, mostly:-) in contributing.html,
I have put changes.jabberd.README and changes.jabberd.patch in
the ftp directory. There didn't seem to be a contrib subdirectory,
so I assume that I must be in the contrib directory when I start the
ftp session.

This is a bunch of diff -c outputs for the changes I made to make
jabberd compile without gnumake and gcc. I basically just got rid of the
non-standard extensions. And while I was there, I added stuff to make it
work for hp-ux, at least hp-ux on 11.x and a PA 2.0 (i.e. 64 bit) machine.
I didn't make the changes described previously for pth, since I compiled
pth 1.4.0 separately, and pth-config used that.

Hopefully, these will get merged into the code...
