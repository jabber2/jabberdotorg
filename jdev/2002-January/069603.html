Admins can send broadcast messages and legitimately send all that fun XML
in the jabber:iq:admin namespace (see how many users are online, etc.).
The broadcast mechanism is really part of mod_admin for the benefit of
server admins so they can inform users that the machine is being rebooted
or whatever -- not intended for general use.
