I've only been on this list for a couple of months, so please excuse any ignorance!

Anyway, I'm interested in writing a web service listener for .NET that can handle SOAP messages over Jabber. However, I can't seem to find any documentation detailing server enhancements that are being/will be made to support Jabber for application communication.
