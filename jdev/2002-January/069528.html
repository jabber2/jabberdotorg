Hello,

The script which generates those .tgz files at download.jabber.org is 
broken.
So, you need to use cvs directly to get the latest msn-t code.  Assuming

you've
got cvs installed, the commands to do this are:

cvs -d:pserver:anoncvs at jabber.org:/home/cvs login
cvs -d:pserver:anoncvs at jabber.org:/home/cvs co msn-transport

The password is "anoncvs".

Once you've got the source, run these commands:

cd msn-transport
./bootstrap
./configure
make

Then follow the README for setup instructions.
