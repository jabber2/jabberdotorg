If you send many messages or long messages in a short time, AOL servers
complain and you will get that "Talking too fast" error. AIM-t just passes
this error through, it does not generate it. The reason the error does not
show up with old AIM-t versions is that they simply ignored/dropped the
error.

With official clients you also get this error if you send very many
messages.

If you push it (keep on sending messages although you get warnings) you'll
eventually be disconnected by AOL. AIM-t does not handle this properly
yet.
