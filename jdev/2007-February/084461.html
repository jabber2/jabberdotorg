Loudmouth 1.2.0 released!

What is Loudmouth
Loudmouth is a small C library for writing Jabber clients written using
GLib. It's designed to be extensible while maintaining ease of use.

Loudmouth 1.2.0 is the new stable release of Loudmouth and is the same
release as 1.1.5 released under a stable version name.

This version contains several important bug fixes compared to 1.0.5 and
everyone is adviced to upgrade to this version.

Feature additions compared to 1.0.5:

    * Support for international domain names, LM-42
    * Improved message IDs, LM-56
    * Improved escaping of outgoing messages, LM-48
    * Memory leaks fixed
