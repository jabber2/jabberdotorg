There was a pre5 release planned for today, but instead of taking the time
to put together a release that will be out of date by Monday, I'd like to
invite anyone that wants to test/evaluate the current 0.7 base to get a
copy via CVS and try it out:
	$ export CVSROOT=:pserver:anoncvs at jabber.org:/home/cvs
        $ cvs login
	Password: anoncvs
	$ cvs checkout jabber
	(list of files being downloaded)
	$ cd jabber/jlib;./autogen.sh && make && make install
	(repeat for jlib, etherx, and jabber-transport)


The current source tree available is stable and leak free, supports
offline messages and rosters, and is ready to be tested in odd build
environments.  Please report any feedback to the right team or to
bugs at jabber.org.

We're still on schedule for the Monday release, and will be focusing on
catching up documentation, clients, and transports after Monday.
