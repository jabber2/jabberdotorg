Folks,

The first beta release of OpenIM 1.0 Java Jabber server available at. 
http://www.javajabber.net/en/

Version 1.0 supports most of basic jabber protocol features.
Persistant storage is done in an XML file, using the Apache cornerstone store 
class. 
It has been tested successfully with GAIM, Kopete, PSI, GreenThumb and Exodus 
client software.
Final 1.0 version is expected for end of july.


OpenIM Java Jabber Server is an open-source Java implementation (Apache 
License) of Jabber Instant Messager.
The purpose of the OpenIM project is to produce a fast, simple, and highly 
efficient instant messager server with high modularisation and a codebase 
that uses the Avalon tools created by Apache's Jakarta project.
