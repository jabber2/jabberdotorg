On Fri, Jul 25, 2003 at 05:15:13PM -0500, Peter Saint-Andre wrote:
> It's been a while since I've used any of the console clients out there.
> Does anyone have a recommendation or now of a project that is actively
> building a really slick CLI client?

I am writting CJC (Console Jabber Client). Currently it is a part of
PyXMPP project (http://jabberstudio.org/projects/pyxmpp), but will be
separated when PyXMPP API stabilizes. 

CJC is not pure CLI interface - it has some GUI, but terminal based and 
it is mostly command driven. It is supposed to be skinnable, customizable
and extensible.

There has been only one, unstable snapshot, release since the project
was started, but it is being actively developed in the CVS. It is even
quite usable now (I use it from time to time for regular jabbering), but
still lacks a lot of functionality (most basic stuff to be implemented
in next few days, I hope).
