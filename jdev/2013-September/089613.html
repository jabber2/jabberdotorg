A New Internet-Draft is available from the on-line Internet-Drafts
directories.


	Title           : Use of Transport Layer Security (TLS) in the
Extensible Messaging and Presence Protocol (XMPP)
	Author(s)       : Peter Saint-Andre
	Filename        : draft-saintandre-xmpp-tls-00.txt
	Pages           : 6
	Date            : 2013-09-09

Abstract:
   This document provides recommendations for the use of Transport Layer
   Security (TLS) in the Extensible Messaging and Presence Protocol
   (XMPP).  This document updates RFC 6120.


The IETF datatracker status page for this draft is:
https://datatracker.ietf.org/doc/draft-saintandre-xmpp-tls

http://tools.ietf.org/html/draft-saintandre-xmpp-tls-00

submission
