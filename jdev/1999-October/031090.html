> This thing is a bitch. I can't figure out how to get the thing to build.

Well I can't speak for the AIM team, but here's my take on it:

get jlib and etherx, and jabber-transport if u like, BUT the
jabber-transport for 0.7 is only a whole lot of stubs, work is currently
being done on jlib and etherx.

get the latest jlib cvs and excute the following commands:
./autogen.sh - runs automake/conf/header and makes a configure script
./configure
make
su
make install

then get the latest etherx and execute the same commands, everything
should build fine... BUT as I said above the jabber-transport doesn't do
anything at the moment, we're still getting etherx to work nicely :)

If u want to play with the aim client etc, get jabber release 0.6 which
has a working server etc.
