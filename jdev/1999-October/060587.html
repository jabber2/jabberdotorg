The 0.7pre2 release is now available:

	http://jabber.org/developers/dist/0.7pre2.jab

It supports authentication and basic message routing between connected
users.  Installation is now easier if you don't have automake/autoconf
installed.

In the next few days the protocol docs will be updated at docs.jabber.org
to reflect the protocol being used in 0.7.

0.7pre3 should be out by the end of the week containing presence and
roster functionality, as well as work on etherx to route messages between
servers/transports.

Join us on #jabber if you have any immediate problems/questions!
