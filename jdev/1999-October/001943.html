On Tue, 12 Oct 1999, Scott Robinson wrote:
> Could you tell me the difference between your system and the many other
> Open Source systems being developed? (I've looked at the docs)

You are right.  There are not too many differences between Jabber and
OSIM.  Mainly the packet vs. XML communication is probably the biggest
difference.  I also hope to have the client chat work more like AIM's chat
(i.e. instead of having seperate windows for each message sent/received,
have a dual pane window that displays the messages sent and received).  I
know this is a client issue, not a server issue....

The goal is to get something implemented quickly, and make it simple.
That means no extra fields in the messages, and no routing capability,
just to name a few.
