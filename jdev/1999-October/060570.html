Finally the day has come, 0.7 is starting to progress towards a release!

Start at: http://jabber.org/developers/dist/0.7pre1.jab
Download directly at: http://download.jabber.org/0.7/jabber-0.7pre1.tar.gz

This is pre1, it is NOT a functional release, only a test of the base and
something for those of you that want to start getting their hands dirty.
I'm planning regular pre* releases every few days to a week, so the pace
is going to pick up and we'll try to get 0.7 out in a few weeks.

If you have any problems and/or questions, please join the appropriate
team and use that list (see above web page).  I'll be updating many of the
team sites over the next few days to reflect the 0.7 issues and be posting
TODO lists for each team.

Thanks,
