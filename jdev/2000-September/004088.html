In case anyone's interested, I just put up a LXR cross-ref of the 
latest jabberd source code at http://pops.dyndns.com/lxr/ This is
running off a box on my home network, across my DSL line, so please
be gentle ;)

I ended up doing this because I was getting frustrated jumping from
one file to the next trying to backtrace function calls to see what
configuration options I needed to change to get stuff working.

Oh, and updates will be sporadic, at best. I expect to update to the
1.1.2 release when it comes out, and keep it up to date on the latest
semi-official release after that, until something more official goes up.

Hope it's of use to some of you,
