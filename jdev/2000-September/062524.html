>    First, is "groupchat" supported by the JabberIM client, 
>     developed by jabber.com ??? I don't see how to use it.

No.. The currently released version of JabberIM does not support groupchat
functionality. Keep your eyes on the website though, as newer version will
probably be released sometime in the near future which does this.

>    What clients do support it ???

Most of the OpenSource clients support groupchat:
  for windows, try winjab
  for macs, try Jabbernaut
  for Linux try Gabber
  for others try Jarl (Perl/tk)
You can get pointers to these clients at the central jabber "end-user" site:
  http://www.jabbercentral.com
  
>    Do all servers support "groupchat" or is it a special feature,
>     which has to be enabled ???

Groupchat is a "transport" just like the aim or icq transports. It's NOT
part of the standard pkgs. You can get the source from cvs at:
  http://download.jabber.org/cvs
  
>    How do you add or remove groups ???
>    How do you allow or disallow people from joining a group ???

Right now, the groupchat stuff is like "private rooms". So the room is
created when the first person enteres the room, and gets destroyed when the
last person leaves. Currently, there are no security features like "ops" and
other IRC type concepts.

Have fun and enjoy.
