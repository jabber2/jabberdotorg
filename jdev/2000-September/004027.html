It does work between clients if you stick to the "semi-standard" of 
using / as path seperator, e.g "copanies/jabber.com" indicates group 
"companies" contains sub-group "jabber.com". I will eventually add 
this to Jabbernaut, and maybe WinJab, Gabber etc. will add sub 
groups, too. But if they add nested groups, they will use the 
standard.
