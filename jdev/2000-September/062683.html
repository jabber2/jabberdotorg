Hi, folks.  I wanted to let you know that the Jabber Tcl project now has a
SourceForge site.  The URL is http://sourceforge.net/projects/jabbertcl and
I'll be setting things up for SourceForge to be the main source of
information on this little project.

If you're interested, there are currently two subprojects to Jabber Tcl.
The first is JabberLib, which is a Tcl library interface to Jabber that
makes writing Tcl/Tk clients much easier.  Built on top of JabberLib is
zABBER, which is the first Tcl/Tk Jabber client.

Both of those were initially written by Kerem HADIMLI, but he doesn't have
time to actively maintain them anymore.  So, I've taken the code he wrote
and will be adding new features to both zABBER and JabberLib.  And any other
contributions are welcome, too, of course.  If you've got an interest in
Tcl/Tk and Jabber, come join us.

So far, I've started populating the SourceForge bug tracker with bugs and
enhancement requests, and I'll import the code into the SourceForget CVS
repository sometime soon, too.


Todd.
