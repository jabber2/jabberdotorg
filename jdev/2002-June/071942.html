I've put together what i think is a roughly releasable headline component. I've
been running it for some time on my own server as "rss.rifetech.com", and now I
hope somebody else can benifit from it.

http://rifetech.com/~atrus/janchor/
http://rifetech.com/~atrus/janchor/README.html

The main features present that I haven't seen elsewhere are a dynamic lists of
news sources (maintained through jabber:iq:reigster), and intellegent presence
(eg, xa/"Failed to parse XML" or dnd/"Failed to conenct to headline source").

I think I've made the configuration more or less straightforward.

the response is good, i'll consider looking into getting it up on
