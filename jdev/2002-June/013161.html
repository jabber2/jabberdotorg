> Well now I'm very confused.  I thought JabberD on windows required Cygwin.

jabberd requires the cygwin environment (specifically the GNU dev tools like GNU make) to *compile*; the resulting executable binary does not.  If you just got the binary, you shouldn't need cygwin -- though it's such a nice package, you might as well have it anyway. ;)
