The current version of the AIM gateway (transport) does not automatically
import your current AIM buddy list.  I'm not sure as to the priority of that
feature in the minds of the AIM transport developers.  I feel your pain.  I
have 200 buddies on my AIM buddy list.  It's not fun to keep them in sync
with my jabber account.  There is a few AIM buddy list import tools out
there if you're an able user.  Off the top of my head
http://www.jabbertools.org/ should do nicely for you.  It's a web based
application that is pretty easy to use.

Hope this helps!
