If you'll use a public jabber server you may get in trouble with the karma
limits. It's a server side bandwidth delimiter that makes it
impossible (if karma is enabled, which is the default with the jabber server
I use) to send a message of half a meg (or even 50k).

All you get is a frozen connection and an "out of karma" warning in the
server logs.

But I assume you'll use your own server, so it won't be a problem.
