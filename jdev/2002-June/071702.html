Try doing:
echo "This is a message" | mail jabberuser at jabber.hisserver.com
That'll work if he's got an SMTP-t installed.
If you want to make sure it always works, install your own SMTP-t and do:
echo "This is a message" | mail jabberuser%hisserver at jabber.yourserver.com

Hope that helped,
Dave Cohen <dave at dave.tj>
