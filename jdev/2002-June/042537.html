This depends on the Jabber client you are using. There should be 
something like "add contact" in the menus. What you need to know to add 
them is their Jabber ID (=JID).

A JID looks like an e-mail address (user at domain). The domain is the name 
of the server, you have choosen it either with the <host/> element in 
your jabber.xml file or with the command line option -h. If you havn't 
choosen anything it defaults to "localhost".
