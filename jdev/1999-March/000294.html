Thanks, I added the fix to the sources. Everybody remember that you can
grab and test the latest version of Jabber from the CVS tree with the
following commands :

cvs -d :pserver:anoncvs at jabber.org:/cvs login

(There is no password, just hit enter.)

cvs -d :pserver:anoncvs at jabber.org:/cvs co jabber

cd jabber; configure; make
