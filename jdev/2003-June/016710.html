Harsha,

First, you'll probably want to choose a Jabber/XMPP Java client library 
to use. A few to choose from are:

  * Smack -- http://www.jivesoftware.com/xmpp/smack/
  * Jabberbeans -- http://jabberbeans.jabberstudio.org/
  * Yaja -- http://yaja.sourceforge.net/

Even using a client library, you'll still want to learn about how the 
XMPP protocol works. The documentation at Jabber.org is a good start. 
Another great resource is the book "Instant Messaging in Java" by Iain 
Shigeoka. You can find the the first chapter of the book for free at:

http://www.jivesoftware.com/products/messenger/book/

Regards,
Matt
