Hi Harsha

Your best bet is find an API that has very good examples (this is key!)  
and documentation. I tried several and found the MUSE API to be the  
best in this regards (http://www.echomine.org/). I Also found the  
developers to be extremely helpful. There are several other API's to be  
found at http://www.jabber.org/. If your going to try do it from  
scratch I recommend reading O'Reilly's "Programming Jabber" and  
"Instant Messages in Jabber" by Lain Shigeoka (ISBN 1-930110-46-4). The  
latter contains the source code for a jabber server in java as well as  
a client, the source code is available from the authors website  
(http://www.manning.com/shigeoka/)

regards

Colin
