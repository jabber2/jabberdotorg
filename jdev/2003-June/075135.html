Ritu Khetan wrote:

> Hello all,
>
>    Is it possible for users on two different jabber servers (running 
> two separate domains) to communicate with each other such that one of 
> the Jabber servers has a static IP address and other gets a dynamic IP 
> address and when it comes online.

You can use a dns name instead of the IP as the name of your jabber
server (in jabber.xml). I use this for testing purposes at home with 
dynamic
IPs and a dynamic IP dns service such as dyndns.org. When I connect, I
get a new IP, but register this IP at dyndns for a dns name.

It is not perfect, but it works.

Hope it helped.

Regards Bert
