issues with Mediaone about my DNS name not always available, so I use the IP
address instead of the name. I can set the name field to either the internal
ip address or the external ip address. The alias field gets the other
address. If the name field is set to the internal address, and the alias
field is set to the external address, only the internal clients can connect.

attribute". When I switch the entries, then the external client connects and
the internal gets the error.
