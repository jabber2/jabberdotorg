Fernando -

The GetPres method should return "nil" or "null" or "nothing" whatever your dev.
environment uses when there are NO presence packets in the presence database for
that JID. This is the intended behavior.

If you get ANY result, it means that the user is available, and the IJabberPres
object will contain information about that connection.
