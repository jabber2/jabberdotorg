I of course can't speak for everyone - but it would probably be best to test
against jabber.org. Being a development server, you can file bug reports on
anything that appears broken or anything you manage to do to break the
server, making the final product better.

However, you might want another server handy for 'second opinions'. I don't
know of any radical server changes that should cause any inconsistancies,
though. (or any planned, for that matter)
