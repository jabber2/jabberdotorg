LLuna 2.3.7 is now released!
============================

What is LLuna?
--------------
LLuna2 is a windows client program for end users wishing to meet other Web surfers on Web sites. LLuna2 users will see other users as an icon on the page while they are on the same Web site. 

Info and download: http://www.lluna.de/

Features:
- group chat with the direct-input bubble chat
- private chat channels
- chat gestures 
- presence status and away message
- Jabber protocol
- digest authentication
- uses the existing distributed Jabber network
- iconic video
- privacy filters
- image avatar moveable on the page
- animated avatars and avatar plug-ins
- personal profiles 
- toplist of URLs with configurable privacy
- find user function
- configurable chat rooms

The client sends virtual presence messages to indicate the user's presence on Web sites and receives information about other users in the same location at the same time. The client displays users directly on Web pages. There is no ordinary application window, only the avatars with chat bubbles. The virtual presence system including LLuna2 is designed to protect the privacy and prohibit any indecent use, be it commercial use, advertising, or profiling. Important measures are: URL scrambling and a distributed architecture without a central registration. 

LLuna was developed by bluehands (www.bluehands.de) for SURFnet (www.surfnet.nl), the Internet provider for higher education and research in the Netherlands. SURFnet sponsored the development to promote Jabber and Virtual Presence on the Web. SURFnet put the code into the public domain under LGPL.
