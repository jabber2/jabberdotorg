My understanding is that the origin of a stanza MAY set an id in the
stanza.  If that is the case, whatever responds to the stanza MUST include
the id in the reply.  In other words, if the client includes an id tag, the
server must return the id tag in the reply.  Similarly, if I as a client
respond to, say, a jabber:iq:version request, if the other client included
an id in their request, I must include it in my reply. :)
