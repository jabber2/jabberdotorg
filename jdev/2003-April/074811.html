Hi !

   New release of wpjabber is available. Mainly WPJ was changed:
a) more stable and functional SSL
     - you can define own SSL algorithms to get faster or more safe 
connections,
     - pings are supported correctly under SSL

b) changed config file
    - removed some unnecessarily config options
 
c) modified log files
   - removed some useless debug informations
   - added logs for session start/send, errors
   - added full config info on WPJ startup

Now WPJ is stable, fast and very safe.

Get it on http://wpjabber.jabberstudio.org.
