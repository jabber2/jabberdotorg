I've been working on a very simple web client based on DHTML and
JavaScript.  It uses Flash for the socket connection or the Winsock
ActiveX control.  It works in IE5+ and almost on Mozilla.

You can see an example at http://workplaceportal.com/jabber/

The example here uses the Winsock ActiveX control instead of the flash
object because with Flash XML sockets, you can only open a connection to
a Jabbber server within the same domain as the flash movie.  

I'd appreciate any feedback.  Keep in mind that it's still work in
progress.
