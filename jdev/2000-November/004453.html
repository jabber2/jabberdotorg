Jabbernaut 0.5.4 has just been released. Go to 
http://jabbernaut.sourceforge.net/download.html to download it.

This new release features compatibility with JServer 1.2 and several 
other improvments, including:

  * JID Blocking to help prevent you from being spammed
  * progresss dialog during login, and timeout for login (60 secs)
  * detection of broken connections improved - you will now get a warning
    if you get disconnected from the server
  * new meta group "my resources" so you can jabber with your other selfs <g>
  * events now can bring Jabbernaut to the front
  * greatly improved "Add User"/"Edit User" dialogs
  * many bugfixes and minor improvements as always

If you use an older version of Jabbernaut, make sure you get this 
latest release!

FYI, a largely reworked UI is in the works, in addition with a usable 
history (incoming&outgoing messages saved on a per-user basis, also 
records status changes etc.).
