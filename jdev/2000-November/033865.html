Hi guys,

    Alot of you have noted that the CVS version of "jabber2" would not build from CVS.. this is largely becuase the HEAD version of CVS is now jabber server 1.3, under development.  if you want to have the jabber 1.2 server, please checkout the RELEASE-1_2 branch, instead of HEAD.  

    FYI, the 1.3 branch now compiles, as long as you have the CVS version of libjabber installed... the biggest changes were the move from io_select to MIO (managed input/output), and soon, the move to pthreads.  untill such time as the 1.4 server is released, you may consider the HEAD CVS branch of "jabber2" to be unstable 1.3.  If you have any further questions, feel free to post them here, or jabber me at tsbandit at jabber.org.

    Keith
