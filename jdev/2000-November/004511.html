Assuming that you are trying to use jabber 1.2, you must rebuild libxode
and libjabber (in that order). There were several changes to libxode,
and both libjabber and the server need to be rebuilt with the new
support.

Also, if you don't install to the same path, you will most likely have
to delete the older installation, and do a complete rebuild (removing
the configuration cache if you built before on the same source tree). I
had files in /usr/local/, and installing into /usr broke because the
configuration scripts used the old version.
