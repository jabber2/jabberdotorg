First, see the configuration 'FIX' email before.

you need to either remove the <host> tag with the command line parameter in it, or
combine the two host lines into

<host><jabberd:cmdline flag='h'>myhost.myserver.com</jabberd:cmdline></host>

What is happening is that the -h flag isn't set on the command line, causing the host to be set to null - which means receive all
messages. It is looking at server-to-server communication and treating it as local communication.
