I don't think you can use non-Jabber clients to connect to Jabber
groupchat, since the non-Jabber clients would need a way to understand
<message type="groupchat"> (suitably translated into the non-Jabber
format via a transport). I too don't use non-Jabber IM services so I
don't know if they even have an interface for groupchat. Perhaps our
esteemed transport developers might have some insights....
