The JUD module in CVS is an old C one that worked with flat files. 
users.jabber.org is running a Perl and MySQL version that is faster. 
Since it is Perl it requires a number of modules that make an install
script a better way to install it.

My plan is to remove the MySQL requirement and allow for you to pick the
storage model, including a flat file that is stored in memory for small
sites but fast access.
