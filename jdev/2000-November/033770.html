JabberBeans (http://jabberbeans.org) now has code suitable for writing 
serverside components in the 1.2 architecture. There is even an included 
example (in src/jabberbeans-tests/simple-case) called 
HelloWorldTransport - it does not provide registration, just a simple 
'send message, get reply' application - but it should be enough for you 
evaluate.
