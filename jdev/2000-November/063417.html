Guy,
  You must make sure that all machines wanting to deal with your server
must know the anems of the server the same via DNS. It can not use just
hostnames.   So if you are using jabber.telemessage.com  then there must be a
DNS entry of jabber.telemessage.com behind your firewall. You do not need to
have your server speaking to the jabber.org servers. (Though it is a bit more
difficult to set things up if you can not)

The DNS is IMPORTANT as currently libetherx-libxode (I do not have source from
here) will only look in the DNS.
