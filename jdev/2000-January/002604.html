>     Yes, this is the error i've got.

  I'd say this is the most common error people are having, and it's not really 
a jabber install error.  It's becouse the linker can't find libpth, which 
jabber tries to link against.  I can't speak for FreeBSD, but under Linux, 
there is a file called /etc/ld.conf that contains the paths the linker looks 
for libraries.  Usually, this will, for some reason, not contain the lib 
directory libpth was installed into, or the user never ran an ldconfig for ld 
to rebuild it's cache which tells it where what library is, and it's version..
