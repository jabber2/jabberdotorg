How about (strongly) recommended switche names and configuration format for
jabber clients? At the very leats iit would be nifty if say all perl clients
could use the same config file, if not all jabber clients...

I propose something like the following:

#jabber Configurations
%default = {
	user=>'bob',
	nick=>'bobo',
	server=>'jabber.org',
	port=>5122};
%other = {
	user=>'dave',
	pass=>'pass',
	server=>'localhost'}
#client specific options
%_PBnJ = {
	quiet=>1,
	log=>0 };


And then for switches:
C	specify which configuration to use

S	server
P	port

u	user
n	nick
p	pass
