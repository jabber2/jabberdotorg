IMified (http://imified.com) had an http API that front-ended some xmpp functions. Internally, it also used xmpp for many of it's control functions. Starting servers, provisioning accounts, and most other things were all done over xmpp. 

IMified's been shuttered for a while now, but our use of xmpp isn't. Our Rayo project (http://voxeolabs.com/rayo/) is a protocol for controlling voice calls that uses XMPP as a transport.
