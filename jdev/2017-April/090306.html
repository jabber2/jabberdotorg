I actually just solved the issue.

The problem was that in the AWS server instance -> security groups 

I just needed to add support for xmpp messages to flow. I just allowed the traffic to flow and All protocols to be used(as there isn't any option for XMPP protocol specifically) and tried to connect from Pidgin and the connection was made.
