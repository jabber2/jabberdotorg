Hi,

After 4 months of work we are proud to announce that Spectrum 0.3.1 is
out. Spectrum is an XMPP transport/gateway written in C++. It uses the
Gloox XMPP library and libpurple for “legacy networks”. Spectrum is
open source and released under the GNU GPL.

As usual, many bugs have been fixed and some new features have been
introduced (key features include support for XEP-0106: JID Escaping
and localization). The full list of changes is available at
http://spectrum.im/projects/spectrum/wiki/ChangeLog. If you plan to
upgrade from older version, you should read Upgrade Notes
(http://spectrum.im/projects/spectrum/wiki/UpgradeNotes) too.

You can read installation instructions for different Linux
distributions and download Spectrum on
http://spectrum.im/projects/spectrum/wiki/Installing_Spectrum.

We're also looking for more translators. If you're interested, please
visit http://spectrum.im/projects/spectrum/wiki/Help_translate.

If you want to help us or just talk with us, feel free to join our room:
xmpp:spectrum at conference.spectrum.im?join.

Regards
The Spectrum Team
