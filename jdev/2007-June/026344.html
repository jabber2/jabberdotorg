Hi all,

I'm happy to announce another major release of gloox, a feature-rich C++ XMPP 
library.
More info at <http://camaya.net/gloox>.

Major new features include:
- full MUC support
- file transfer (XEPs 0065, 0095, 0096)
- XPath support
- HTTP and SOCKS5 proxy support

You can find the full ChangeLog at <http://camaya.net/glooxchangelog>.

Jakob
