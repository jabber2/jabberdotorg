Hi All,

I've managed to make my Yahoo transport code more or less stable. It is available in the xmpppy project cvs on http://xmpppy.sf.net


Features:
	Jep-0100 gateway interaction is the basis of the semantics.
	Jep-0093 Roster item exchange for Yahoo roster items at registration.
	Jep-0008 Avatar support
	Jep-0045 MUC for yahoo global chat rooms (this is not the Yahoo/Lycos chat as used in the UK)

Requirements:
	Python >=2.2
	Python XMPPPY library
	Curphoo code for Yahoo auth hashes (included in CVS)

Any bugs to yahoo-transport at hopeless-newbie.co.uk please.
