A couple of quick notes for those are are deploying any of the above code:

Yahoo-t 2.3.2: Make sure you set the Yahoo! server in the configuration
to 'msg.scs.yahoo.com'. I did not change the example configuration in
the release, so it still has the old host name. Also, if you are using
the Yahoo! transport with jabberd 2.0, you will need to subscribe to
your Yahoo! contacts manually, as the automatic subscriptions for your
Yahoo! contacts does not work.

JCR 0.2.2: Try to use the 'main.c' provided with JCR, rather than the
one in the mu-conference 0.6.0 release. The 'main.c' has changed quite a
bit, and the reconnections to the jabber server will work correctly if
you use the JCR 'main.c'

You can also use JCR 0.2.2 with the Yahoo! transport. The current Yahoo!
transport release (2.3.2) has a 'jcomp.mk' and a 'main.c' that is
designed to work with JCR 0.2.2.

Have fun ...  questions and bugs to the lists, please.
