Hi!

I would like to announce new release of Tkabber 0.9.7.

Release highlights:

     * Updated support for file transfer (JEP-0095, JEP-0096, JEP-0047,
       JEP-0065)
     * Support for colored nicks and messages in conference
     * Better multiple logins support
     * Updated support for xml:lang
     * Support for IDNA (RFC3490)
     * Many fixes and enhancements

New release is available here:
http://www.jabberstudio.org/projects/tkabber/releases/view.php?id=621
