jabber.ru is using ejabberd server (Developped in Erlang) and I suspect
that this is the same for XMPP.ru.
Our own Jabber server (erlang-projects.org) also support pubsub (Ejabberd
deamon). You cannot register through Jabber but need to connect on the
portal metafrog.erlang-projects.org. This is mainly to attract users from
the Erlang community.

Pubsub is enabled by default on the Ejabberd version of the server, this
is why server using ejabberd are very likely to have pubsub enabled.
