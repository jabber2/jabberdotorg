We've had quite a bit of discussion recently about the "IO Data"
proposal, which is a protocol for web services over XMPP:

http://www.xmpp.org/extensions/inbox/io-data.html

As a result, I have created a new email list for discussion of web
services and other machine-to-machine communication using XMPP technologies:

   http://mail.jabber.org/mailman/listinfo/ws-xmpp

   mailto:ws-xmpp-subscribe at xmpp.org

If you are interested in these topics, feel free to join the list. We'll
try to hold an associated groupchat in the near future.
