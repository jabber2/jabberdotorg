One day later than I wanted, but the second release candidate of 
jabberd14 1.6.0 is now available.

As with the first one, testing and feedback is welcome.

Changes since the first release candidate:

- Improved installation compatibility with RedHat.
- Administrator sees s2s connections when doing service discovery by
   default.
- Adding certification authorities bundle to the distribution.
- Loading certificates including intermediate CAs needed to verify them.
- Compatibility improvement with mu-conference for roster subscription
   syncronization.
- Sending unavailable (available) presence if contact is added (removed)
   from the active privacy list. Immediate invisibility when updating, or
   changing privacy list.
- Added configuration option to mod_xml (Private XML Storage) to be able
   to increase compatibility with clients not understanding XMPP stanza
   errors.
- Fixed typos.
- Now including French and Dutch translations.

The download is available at http://download.jabberd.org/jabberd14/

Thanks for all feedback I got so far.
