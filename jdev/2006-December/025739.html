>
> Hi,
>
> I'm developing a Jabber client in VB.NET for a school project and could
> use a little advice.
>
> 1. Roughly how complex of a project is this? I have about 9 weeks to work
> with. Any advice/warnings/suggestions for a newbie? Or am I just biting
> off more than I can chew?

Before taking on a project like this, you should be somewhat
experienced in XML and client-server application development.  If you
are new to these concepts, I would recommend using a pre-existing
library to get you started (if you can).

> 2. Can anyone recommend a good XML parser to use with .NET?

The .NET core provides a lot of functionality in the System.Xml namespace.
Perhaps this article may help:
http://msdn.microsoft.com/xml/default.aspx?pull=/library/en-us/cpguide/html/cpconemployingxmlinnetframework.asp

Good luck!
