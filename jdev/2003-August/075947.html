The message sent should not be blank, which is strange indeed.

When you set the log4j.properties, by default it outputs the logs and XML
data to the console (ie. your dos prompt window, your IDE console output
window, your linux terminal window, etc.).  Try changing the first line's
log4j.rootLogger to

log4j.rootLogger=DEBUG,console

And see if you see any output.

Also make sure that you are using the newly compiled muse library and that
you have aspectjrt.jar in your classpath.

Let me know if it works and if you see anything XML outputs.

Good luck,
Chris
