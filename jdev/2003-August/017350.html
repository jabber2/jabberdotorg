There is a new project out at jabberstudio.org 
(http://www.jabberstudio.org/projects/pcj/project/view.php) that 
encompasses PoCo::Jabber and related tools for the POE/Perl developer. 
Sources have been checked in from the latest versions and now 
development will commence against that tree for the various libraries. I 
am looking for developers and testers (and the usual assorted crew of 
people all OSS projects need, but never get).

To begin working this project it is imperative to first look at POE 
(http://poe.perl.org) to understand that these libraries work within the 
POE framework, and how to best leverage them.

The libraries are also available at your local CPAN mirror, if you 
prefer to install the releases that way.

Thanks for taking the time to read this and I hope at least some 
interest will be generated by this :)

Nicholas
