Maybe it is of interest for some people on this list as well. I just set
up a public installation of jabberd2 (CVS version) on the domain
amessage.co.uk.
The installation is intended to provide me with experience of jabberd2
field operation *g*, get experience with jabberd2 compatibility and
allow users to see what is coming in the new server generation.

At the moment only a base installation of jabberd2 is running (no
transports connected to jabberd2) and mysql is used as authentication
and storage backend.
