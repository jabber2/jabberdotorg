Try reading the README file of the Muse package distribution.  Inside, it
will tell you how to enable debugging inside muse.  Simply compile the
debugging into muse and then you will be able to see the incoming and
outgoing XML messages outputted (You will need to configure log4j.properties
file as well).

Then you can see if the message is sending the proper XML data.
