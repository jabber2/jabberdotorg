Your question is not clear but I assume you are planning to develop
flash based Jabber client. I would suggest to use XIFF library
(http://www.jivesoftware.org/xiff/) but it's in Action Script not in
Java.

If you wanted to develop Java based client then my bet would be on 
Smack( http://www.jivesoftware.org/smack/ ) and 
JSO(
http://www.jabberstudio.org/cgi-bin/viewcvs.cgi/cvs/jso/example/EchoClie
nt.java?view=markup ) Libraries.

For more details explore Jabber.org
(http://www.jabber.org/software/clients.shtml)
