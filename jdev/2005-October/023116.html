We are pleased to announce a new release of the Open Source Jive Messenger 
server.

The new release includes an equal number of improvements and bug fixes. Many 
server-to-server issues have been fixed and the admin console is now 
available in Spanish.

Complete list of changes can be found at: 
http://www.jivesoftware.org/builds/messenger/docs/2.3.0.beta1/changelog.html.
You can download the server from http://www.jivesoftware.org/messenger.

Please post beta feedback in the support forum.

Enjoy!

Jive Messenger Team
