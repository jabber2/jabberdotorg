Hi All

tweeze-0.44_00 is available.

Tweeze is now a functional (though not fully complient) client-side 
implementation of XMPP-CORE. The tweeze library can be used to build 
java applications that interact with Google Talk or any other XMPP 
complient service.

You can download from the project page at:

    http://www.ipster.org/tweeze
	
Enjoy,
