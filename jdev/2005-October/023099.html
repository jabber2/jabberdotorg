Hi All

ruse-0.10_00 is now available.

ruse is a proxy server - written in java - that enables jabber 
applications that don't yet support TLS, SASL, Resource 
Binding, etc, to connect to strict XMPP complient services 
like Google Talk.

You can download from the project page at:

    http://www.ipster.org/ruse
	
Enjoy,
