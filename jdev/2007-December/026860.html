I've just put together a very quick module in Python for handling
RFC-4622 URI's. Currently it only supports about 30% of the spec, but it
was enough to add the functionality that I initially started writing it
for (better XEP-0147 support for Gajim). I will be working on it as time
permits but anyone who is interested in contributing is welcome to
download it and share patches.

For the time being you can download it and get a bit more info at:

http://www.the-love-shack.net/2007/12/12/rfc-4622-implementation-in-python/

I expect that it will make it into the subversion repository for Gajim
(http://www.gajim.org) soon, however, so it will be a bit easier to work
on from there.
