Hi!

I have a question w.r.t. messages to unsubscribed contacts.
This is the situation:

User A sends a message to the bare JID of user B which is not subscribed.
User B answers to the full JID of user A. First question: Is the full JID
of user B transmitted in that message? (I guess it is).
Ok, then user A wants to reply to user Bs reply, should he send
the message to the now known full JID of the unsubscribed user B?
Or still send it to the bare JID? What if user B went offline? Does user A
have a chance to get to know about the unavailablility of user B?

I've read about the recommendation in 3921bis that in those cases
directed presences should help, but what if they are not available?
Should messages to unsubscribed contacts generally go to the bare JID?
