The jabber.org admin team has moved all of the existing jabber.org and 
jabberstudio.org mailing lists (and mail infrastructure) to a new 
server.

1. Existing mail settings have been preserved as much as possible, 
including digest vs. regular and "nomail" settings for reading list 
posts via Gmane.

2. List URLs have changed and are now all like this (pointers are being 
updated):

    http://mail.jabber.org/mailman/listinfo/jdev
    https://www.jabber.org/jdev/

3. If you were using jabberstudio.org for mail, your .forward file was 
preserved. However, you will no longer be able to read mail via your 
shell account or IMAP. Contact me if you were depending on a 
jabberstudio.org email address.

4. If you have any questions or experience any problems, please contact 
me (see below) or send us a message via 
<http://www.jabber.org/jsf/info.php>.

My apologies for any inconvenience.

Peter
