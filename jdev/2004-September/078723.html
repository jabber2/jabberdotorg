    At PubSub.com, we've opened up the beta on our Firefox extension
that provides a Sidebar which notifies you in real-time about updates to
RSS and Atom files that you are interested in or postings from any of
over 3 million blogs which match keyword searches that you define. This
is the Firefox version of the PubSub Sidebar for Internet Explorer that
we announced a few weeks ago.
    The PubSub Sidebar uses XMPP as extended by JEP-0060 PubSub to send
Atom entries to the Firefox client software. We're hoping that people
will give this a try and that they'll see that Jabber/XMPP, once
extended by JEP-0060, can be much more than just a chat and file
transfer protocol.
    If you develop a Jabber client, we would very much appreciate it if
you would consider building support for JEP-0060 into an upcoming
version. In any case, please give the PubSub Sidebar a try if you are a
Firefox user and let us know if there is anything we can do to improve
it.
    Developer documentation can be found at:
http://pubsub.com/developers.php
 <http://pubsub.com>  
    Get the PubSub Sidebar for Firefox from:
http://pubsub.com/sidebar-firefox.php
 
        bob wyman
