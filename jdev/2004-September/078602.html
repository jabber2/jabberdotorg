Jesper Krogh wrote:
> Could it be possible to localize it? Something like the
> Debian.org-pages. If you're building it from ground, is it probably
> easier to do now that as an addon later. 
We could but I don't know if we want to. It's hard enough to find people 
that want to help with creating the main site. If we want to localize it 
we either need really active translators (to keep everything up to date) 
or a _really_ good localization system. Adding localization does add a 
pretty big layer of complexity, also when it comes to developing 
applications (like a client/server database).

So I've been planning to just create a site in english but of course the 
licence on the content will enable sites like jabberpl.org etc to take 
documents of this community site and translate it. Then it will be their 
responsibility to keep that up to date.

Of course having a central site available in different languages is a 
big plus but IMO that will be too big a task. IMO it's smarter to add 
that later when the community is larger and we have more resources.

Bart
