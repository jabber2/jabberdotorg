I proud to announce first stable release of xmpppy library.
Features:
- users of jabberpy will find very similar API but much greater flexibility 
and functionality.
- TLS/SASL support, works good with all known to me jabber-servers.
- direct/rerouted/proxied connections, easy-to-add any other way of connecting
- x-data, privacy, disco support
- working. Several small programs writted with use of this library already and
   two transports also use it: irc and yahoo (both by Mike Albon)
Anti-features:
- documentation lacking. Only one howto and two examples of programming 
supplied with the librarty

Homepage:
http://xmpppy.sourceforge.net/

P.S. xmppd.py server package is currently under development.
