Hello all,

If anybody is interested just FYI.

I work on open source Jabber/XMPP server implementation.
It is not mature software yet and not very feature reach but a few weeks 
ago I installed live version for tests on my home server.

During last a few weeks I used it as my main jabber server for communication
with my mates. That was not very heavy usage but thanks to these tests 
I fixed a lot of bugs and improved many things.

Now I am looking for people who could help a little with tests.
People who could spare a few minutes trying to use the service and share
their opinions, send comments and suggestions.

If you have a few minutes give it a try. 
XMPP service server is: tigase.org
Project web site: http://www.tigase.org/
Development and sources site: http://server.tigase.org/

Server offers full RFC-3920 and RFC-3921 implementation including
privacy lists. It also offers offline message storage, non-SASL auth,
in band registration and.... that's it.

And there are many ways how you could leave your thoughts about
the server. There is a discussion forum on website, contact page,
article comments or you can send me message directly by e-mail 
or xmpp to: kobit AT tigase.org
