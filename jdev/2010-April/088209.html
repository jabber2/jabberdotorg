Hi,

We've just released Spectrum 0.2. Spectrum is an XMPP
transport/gateway written in C++. It uses the Gloox XMPP library and
libpurple for “legacy networks”. Spectrum is open source and released
under the GNU GPL.

This release is mainly about improving Spectrum stability, so various
crashes are fixed. New features of this release include NickServ
support for IRC networks and stability improvements to file transfers.

You can check all changes since version 0.1 in the 0.2 Release Notes:
http://spectrum.im/releases/0.2/
Spectrum can be downloaded from: http://spectrum.im/download/

Also feel free to talk with us about your opinions in our room:
xmpp:spectrum at conference.spectrum.im?join.

Regards
The Spectrum Team
