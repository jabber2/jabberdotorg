On Mon, 27 Sep 1999, Jeremie wrote:
> => Connections (client to server and server to server) are XML Streams
> (see protocol section at http://docs.jabber.org/).

Using namespaces, other protcols could be interleaved with Jabber. That
could prove interesting. What would the jabber server do with an
<a href="http://www.jabber.org/" xmlns="http://www.w3.org"> tag? The
options I see are 1) Eat it. 2) Pass it on to the recipient of the last
message. 3) Require a jabber:to="somebody at jabberserver.com" attribute, and
send it there. I prefer option three.
