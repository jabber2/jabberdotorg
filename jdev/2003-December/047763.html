I found res_query in /usr/ccs./lib/libbind.a on AIX 5.1. Configure will
first look for libbind before looking for libresolv.

#lslpp -w /usr/ccs/lib/libbind.a
  File                                        Fileset               Type

----------------------------------------------------------------------------
  /usr/ccs/lib/libbind.a                      bos.rte.net           File

Hope this helps.
