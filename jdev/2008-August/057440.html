You will need to run a cluster where you have a C2S connection manager 
in the DMZ connecting back into the core server inside the intranet (for 
external client connections) and also a S2S component running in the DMZ 
connecting back into the core server (if you want to enable server to 
server federation).

Or just simply relocate your XMPP server so that it is inside the DMZ 
rather than the intranet.
