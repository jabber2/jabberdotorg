Thanks for pointing that doc my way.

I'm going to fork Net::Jabber and XML::Stream to implement the new
component protocol.  I'm not sure how backwards compatible it'll end up
being, so I'm going to fork.  If it ends up not being to difficult, I'll
probably merge my changes back into the main modules and add some switch
to object instantiation to allow use of the J2 protocol.

If anyone is already working on such a project, I'd appreciate it if
they shot me an email.

It seems that the major changes will be in the xmlns handling,
handshaking, and adding a Bind method.  If anyone's interested, I'll
keep you guys posted on my progress.  I'm going to be upstreaming all
the changes I make to the modules.  Also, I've got approval from
management to GPL the logging component that I'm going to write, so you
guys should expect a full fledged logging component for J2.
