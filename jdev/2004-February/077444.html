>
> This might be a great idea.  Can you give a little more detail on how to
> do this?  What does my server need?  When you say the old Sun user logs
> in, just what do you mean?  He still has his desktop but now has an
> extra window running this other connection?

  Exactly.  Pick one new system where you can compile Psi (or whatever,
that's what I use).  Then people open a new xterm, type "xhost
+servername", telnet to the new server and log in as themselves, on the
new server type "setenv DISPLAY mysystem:0.0" and then type "psi &".
There are better/faster/more secure ways of doing this, but those steps
are the simplest, and should work with an X windows implementation of any
age (I was doing this in 1990 on DECStation 3000s).
