Hello all,

FOSDEM is a 2-day conference in Brussels, Belgium for Free and Open Software
Developers. It takes place on 21 and 22 February, entrance is free.

For this event, I've set up a Jabber server to get visitors become acquainted
with Jabber, and communicate in various group chat rooms before and during
the conference. I am encouraging people to do live reports in these
group chat rooms, so if you can't make it to Brussels, you can still
see what is going on over there by joining the chat rooms, or reading
the logs on the website.

The address of both the jabber server and the website is: jabber.fosdem.org.
The general conference room is hallway at conference.jabber.fosdem.org.

You can read the full announcement here:

  http://lists.fosdem.org/pipermail/fosdem/2004-February/001762.html

Also, there'll be a Jabber Software Foundation (JSF) stand on the conference.
If you want to help out there, let me know.
