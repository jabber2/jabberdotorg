jabber.el 0.4 has been released. jabber.el is a Jabber client running
under Emacs, currently only GNU Emacs 21. jabber.el can do basic chat,
service discovery and interaction with services. There is also basic
groupchat support.

Find it here: http://www.intellectronica.net/emacs-jabber
