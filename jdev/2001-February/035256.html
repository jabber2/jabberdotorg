Imjay (http://imjay.sourceforge.net) uses asymmetric key encryption
in its messages.  It uses Diffie-Hellman by default, but the
algorithm is pluggable through JCE.  It hasn't seen any active
development for the past few months, but I'm planning to get moving
on it again soon.  It stores public keys on the server in the user's
vcard profile and the private key in a local configuration file.
