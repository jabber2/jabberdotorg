Maximizing Simultaneous TCP Sockets

A new mailing list has been created,
http://mailman.jabber.org/listinfo/sockets or sockets at jabber.org. The
focus of the list is to discuss the work going on for software
solutions as well as OS configuration and TCP tweaks.  One of the
important new efforts being started is a project to push Linux to the
limits, see how many simultaneous sockets it can support.  This is
going on in partnership with http://www.osdlab.org/, and will be using
the /dev/poll patch (
http://www.citi.umich.edu/projects/linux-scalability/patches/) for
efficiently processing socket traffic.  If you would like to be
involved in this project, have OS-specific tweaks or software you
would like to share,  or would like to start another project for other
platforms (such as using kqueue for bsd), please join the list and
share your thoughts.
