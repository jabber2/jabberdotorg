I had the same problem (with icq-transport). I got icq-trans(cvs) 
to compile and work by doing the following steps:

1. download
http://download.jabber.org/dists/1.4/final/icq-transport-0.9.tar.gz
2. unzip+untar it into the jabber-1.4 dir
3. download http://download.jabber.org/cvs/icq-transport.tgz
4. unzip+untar it into the icq-transport-0.9 dir, overwriting 
   the old files
5. type make
