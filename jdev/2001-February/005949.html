1.4 Release Candidate available

You can snag the 1.4RC base tarball here:
http://download.jabber.org/dists/1.4/RC/jabber-1.4RC.tar.gz

There are bound to be a few build issues or problems that we've
missed, so please report back any issues to bugs at jabber.org and/or the
jdev at jabber.org mailing list.

If we can get any of the problems encountered in the RC fixed right
away, we'll be building the final packages this weekend :)
