Please come and try out a preview of the new GO Messenger. http://im.go.com

To use it you will first need to have an existing web account on one of the
Disney
family of sites, ESPN, ABC, GO, etc.  Or you can create a new account here:
https://register.go.com/go/chooseAge

Right now we have tabs for instant messaging and free email.   We are
working
on tabs for instant sports scores, news updates, etc.

As of this time, we don't have any transports working, but we should have a
few
working early next week.

There are some features that you might have gotten used to like, ignore
lists 
and last login times, etc.  Right now the Jabber commercial server doesn't
support
these, so even though the client supports them, it can only give an error
message
when they are accessed.   We are told that they will be supported in the 
commercial server soon.

Feel free to add me to your buddy list: rwtemple at go.com.  You can also 
add the robot: chatbot at go.com

If you like you can also change the server that the client points to.  Just
go into
the options from the menu and from the connection sheet, change the host
that 
you connect to.

Enjoy,
Robert Temple
