The Source for JabberCOM is licensed under the Library GPL (LGPL) and can be
found at the winjab sourceforge cvs site:

http://sourceforge.net/cvs/?group_id=5039

You'll need the XML directory and the JabberCOM directory.

I'm looking at releaseing the current cvs copy sometime this week. This
version dramatically improves the stability of the worker thread, fixes the
problem of events coming from the "other" thread and fixes many bugs. The
events being fired from the main thread should solve many of the problems
that have been previously posted to this list. Stay tuned.

Peter.
