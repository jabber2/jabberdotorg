I'm building a new in-house CVS mirror for Open Source projects, are there any
provisions to do 'cvsup' style mirroring of the CVS source tree from jabber.org?

Right now I'm set up to do a true CVS mirror of FreeBSD and OpenBSD, a rsync
mirror of CPAN, and normal CVS checkouts of the Jabber.org and SourceForge
trees for the various projects.

I'd prefer to do a true mirror of the jabber.org CVS from which I can browse
with CVSWEB and checkout specific releases on my internal build hosts, instead
of having my mirror server do the checkout.

network (500 developers, 25,000+ employees), I'm campaigning to allow public
access to at least anoncvs and FTP, if not full CVSWEB and CVSUP access.
