Hi,

There are number of choices over there to select from.
If you are doing a research project I would suggest something based
on the technology you are familiar with.
There are libraries ready to use in almost any language you like:
http://xmpp.org/software/libraries.shtml
There is also a long list of servers available written in languages from
C, C++, Java, Erlang, Perl and so on:
http://xmpp.org/software/servers.shtml

If you are familiar with Java I can recommend Tigase project:
http://www.tigase.org/
We have a server written in Java and also libraries for
writing web clients using GWT. More details can be found on the
website and you can also ask your questions using Minichat client.
