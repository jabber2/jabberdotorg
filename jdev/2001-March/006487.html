Contributing to server development

I just set up http://download.jabber.org/contrib/ and (thanks to
stpeter)
a quick outline on docs.jabber.org explaining how to contribute.
I want to encourage anyone with any patches or other server
contributions since the 1.4 release to post their files, and
we can use this area as the starting point for getting changes, fixes,
and enhancements worked into a 1.4.1 release.
If you've been playing around with the 1.4 codebase and think you've
got a pretty good handle on it, and would like cvs access, let us
know.
We definitely could use help in maintainence, performance
enhancements, and new extensions to the server and welcome anyone
willing to assist us.
