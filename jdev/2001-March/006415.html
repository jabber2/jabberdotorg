Silviu,

    Hi, I'm not sure how to answer your question directly, but here goes... =}

    The Jabber Protocol, since pre1.0 has always supported a subscription method
    of gaining another users (or any jid's) presence.  (this requires authorization)

    also, There will probably not be a jabber server 1.5 or 1.6... current development
    will remain for some time on 1.4.x improvements, and more components, before
    moving to the 2.0 "pthreads" server architecture. (pthreads being only one of
    the many major improvements that the 2.0 server will bring)

    Hope this helps!
