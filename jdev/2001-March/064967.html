Presently, crypto support is limited to sending encrypted messages.
There is no support for encrypting blocks of XML, and having
them parsed by receiving clients after decryption. Clients receiving an
x element with the namespace defined in this document should decrypt the
message, and then parse it as it would parse any XML that it received
by normal unencrypted transmission.
