this is an updated version of the second half of the old Jabber
Technical White Paper (which will be deprecated as soon as I can created
some updated architecture docs). I believe this document reflects almost
everything that is supported by the 1.4 server, but I might have missed
some stuff. Let me know what else you'd like in the document (e.g., more
examples, DTDs, whatever) and I'll endeavor to put out a new version
soon.
