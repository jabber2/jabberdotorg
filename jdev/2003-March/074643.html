Jabber has roughly three kinds of "things" it connects to: clients,
other jabber servers, and components.
 
Clients are usually the instant messengers you see on your desktop and
connect via port 5222.  Other jabber servers connect using 5269, and
components connect to whatever port you've told jabber to allow them on.
 
For each of these they use a different namespace.
 
>From the details you gave below, you mean to use the server namespace on
port 5222, which is doomed to fail.  Also, the secret is actually a
password known by the server that the component uses to validate itself.
Clients don't use secrets, and instead have another similar but
different validation mechanism.
 
Do you really mean to program a jabber component?  Or do you want to
write a client?
