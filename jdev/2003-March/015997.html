Hello all. I posted a module at the contrib site that may be helpful to you if
you're interested in searching your Jabber users in LDAP.

This module was templated off the JUD (Jabber User Directory) module.

The JUD module uses a 'global.xdb' file to store user information: name, first, last,
nick, and email.

This module is similar except that the user information is obtained from an anonymous
LDAP search, rather than a localized file on disk.

If you're using my ldapauth2 module to authenticate your users, then this module should
prove useful to you as well.

Send me your comments/questions.
