On Monday, March 10, 2003, at 05:27  AM, sonic Beez wrote:

> I'm trying to compile Jabberd 1.4.2 on Macos 10.2.4 but so far no 
> success. So far I'm having problems with pth. In the updates file it 
> says:
>
> To compile the included pth, first run the command 'cp 
> /usr/libexec/config.* jabberd/pth-1.4.0/' before running ./configure.
>
> But I have no files that match /usr/libexec/config.*
>
> running ./configure just gives me errors about pthreads command not 
> being found.
>
> Any help on this would be appreciated.

I'm pretty sure I used the following when I built 1.4.2 to test my 
development against.

/usr/share/libtool/config.guess
/usr/share/libtool/config.sub

Unless this has changed in 10.2.4, you'll also need libdl, which you 
can get from the Fink project as the dlcompat package.
