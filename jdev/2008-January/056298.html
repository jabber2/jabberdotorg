Hello everybody,

I'm sending this message to inform you that the first beta version of 
Asterisk 1.6 is now available : http://www.asterisk.org/

Asterisk is an enterprise grade IP-PBX, intended to support real-time 
signalling and media communication protocols (SIP, H323, PSTN gateway 
function, ...). Asterisk has a XMPP module that covers both client and 
component connections to XMPP servers, and a GoogleTalk channel driver 
that allows to place and receive calls to/from GoogleTalk clients.

This new release comes with a Jingle channel that we try to keep close 
to the specs.  We'd be glad to have the XMPP developers community help 
us improve our XMPP modules and extend Asterisk's interoperability!

To get Asterisk's source code : 
http://www.asterisk.org/developers/get-source
To report bugs : http://bugs.digium.com/main_page.php

Looking forward to meeting many of you at the next Devcon!

Philippe
